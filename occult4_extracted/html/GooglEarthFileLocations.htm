GoogleEarth file locations

When Occult draws data on GoogleEarth, the corresponding kmz file can be found in the following locations:

Predictions - sourceLocation
Asteroid predictions - path location\AutoGenerated Asteroids\AsteroidPath.kmz
Solar eclipse predictions\Predictions\SolarEclipse.kmz
Lunar predictions - occultation region\Predictions\LunarOccultationRegion.kmz
Lunar predictions - graze path\Predictions\LunarGrazePath.kmz

Observations - sourceLocation
Asteroid observations - site locations\Asteroid\AsteroidSites.kmz
Lunar observations - site locations\Observations\LunarSites.kmz
Lunar observations - plot of lunar limb at the time of the various events\Observations\MultiEvent Moon Limb Plots.kmz
Archive editor - plot of current site location\Observations\ArchiveCurrentSite.kmz
Archive editor - plot of old and new site location\Observations\Archive_OldNew_Sites.kmz
