Constellations - files for boudaries and identification

The files here are used to draw the boundaries of constellations, and identify the constellation for a speciffied RA and dec. The original paper is given here., and the dataset can be downloaded from here.

ConstLinesNS.dat, ConstLinesEW.dat

These two files contain the co-ordinates of the start and end of the lines defining the borders of the constellations. The lines are those of the IAU 1930, and are specified in the equinox of 1875. [At that equinox, the lines are coincident with the lines of RA and declination]. The files are structured as follows:

Data in each file is held as variable-length ASCII data, with CRLF between each data element. The data in the two files are:

ConstLinesNS.dat

380 records. Each record consists of:

  • Declination of North end of line
  • Declination of South end of line
  • Right Ascension of line

ConstLinesEW.dat

397 records. Each record consists of:

  • Right ascension of west end of line
  • Right ascension of east end of line
  • Declination of line

Right ascensions are expressed in decimal hours; declination in decimal degrees.

To plot a line of the constellation boundary, it is necessary to precess the line to the equinox of the chart being drawn. That transformation is non-linear along the length of a line - so that the transformation will generally need to be done for a series of points along the line [especially for long lines.]

Constell.id

This file contains data to enable the identification of the constellation, for any specified co-ordinate. The co-ordinates must be referenced to the 1875 equinox.

357 records. Each record consists of:

  • West right ascension
  • East right ascension
  • Declination
  • Constellation number

To identify a constellation:

  • scan through the file to the first record having a declination <= the specified declination;
  • from (and including) that point, scan to the first record where the East right ascension <= the specified right ascension;
  • from (and including) that point, scan to the first record where the West right ascension > the specified right ascension.
  • Repeat steps 2 and 3 until a record is found where the record satisfies both 2 and 3.
The Constellation number for that last record identifies the constellation.

Constell.nam, Constell.abb

These two files contain, respectively, the full names of the constellations, and the three-letter abbreviation of the name. Data is ordered alphabetically by full name, and in the ordered referenced from the Constell.id file.
