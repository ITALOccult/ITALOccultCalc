Hipparcos Index.dat

This is a random access file with record length 4, containing 120500 records. Each record corresponds to the Hipparcos star number having that record number. The record is a Long containing the record number (unary based) in Tycho2.dat file for that Hipparcos star.

Note that while there are 120500 records in this file, there are only 117953 valid entries in the file. The other entries are set to zero; they correspond to the 263 entries where the Hipparcos catalogue does not have precise positions, and to other non-existent entries in the catalogue.
