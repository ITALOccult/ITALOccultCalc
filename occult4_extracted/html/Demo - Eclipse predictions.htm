Demonstration of Eclipse predictions

Follow the steps set out below to get an overview of the functionality provided by Occult.

Solar eclipse predictions

  • From the Main menu select the Eclipses and Transits tab
  • Click Solar Eclipse predictions On the displayed form there will be
    • a list of solar eclipses for the next 10 years, commencing with the current year - with the first eclipse selected
    • a world map showing the eclipse path (if any) for the selected eclipse.

  • Select the eclipse of 2012 Nov 13. The eclipse path will now be displayed on the map.
  • Place the cursor on the central eclipse path where it crosses the top of Australia, and double-click. A more detailed map, centered on the cursor location appears.
  • In the Plot Sites box at the top of the form, select Australia.site. Small circles will appear on the map. In the adjacent box, select All - and the site names will appear next to the circles.
  • Place the cursor somewhere near the edge of the map and double-click. Notice how the map is redrawn centered on the cursor location.

  • [If you have GoogleEarth installed on your computer, and have indicated this fact in the User Settings (found on the Maintenance tab)].  Place the cursor anywhere on either the world map or the detailed map, and press the right-click button. Select View in GoogleEarth. GoogleEarth will now open, and the eclipse path and limit lines will be displayed.

  • In Occult, on either the world map or the detailed map, place the cursor at a location of interest, and press the right-click button. Select Local predictions. The displayed form gives a detailed prediction for that location.
  • Now place the cursor in a region covered by the loops at the beginning or end of the eclipse region (for example, at Darwin or Alice Springs), right click and select Local predictions again. In the prediction note the extra line that gives the time of sunrise (or sunset) at that location, and the magnitude of the eclipse at that time.

  • Place the cursor anywhere on either the world map or the detailed map, and press the right-click button. Select MultiLocation predictions. At the top of the form, Select Australia.site. Predictions for all sites in that site file are now displayed. You can generate predictions using any site file. In particular, you can create your own site files - using the Maintenance tab, Edit Site files.

  • Place the cursor anywhere on either the world map or the detailed map, and press the right-click button. Select Path coordinates. At the top of the form, set West to 135, East to 155, and Interval to 2 deg. Click Compute to generate the path coordinates.

  • Place the cursor anywhere on either the world map or the detailed map, and press the right-click button. Select MultiLocation predictions. At the top of the form, Select Australia.site. Predictions for all sites in that site file are now displayed. You can generate predictions using any site file. In particular, you can create your own site files - using the Maintenance tab, Edit Site files.

Lunar eclipse predictions

  • Select the Eclipses and Transits tab
  • Click Lunar Eclipse predictions. On the displayed form there will be
    • a list of lunar eclipses for the next 10 years, commencing with the current year - with the first eclipse selected
    • a list of the eclipse contact times
    • a graphic representing the path of the moon through the Earth's shadow
    • a series of graphics of the earth, showing the hemisphere from which the moon is visible at each of the contact times and mid-eclipse.
  • Select different eclipses in the list to see how the prediction changes between eclipses
  • Change the decade of the list of eclipses by clicking the up/down control next to the year.

Transit predictions

  • Select the Eclipses and Transits tab
  • Click Transit predictions [Mercury & Venus]. On the displayed form there will be
    • a list of transits for the century 2000 - with the first event (2003 May 7) selected
    • an image of the Sun, with the black line showing the path of Mercury across the Sun
    • the geocentric prediction; and
    • two images of the Earth - which show where the Sun is visible from at first and last contact.

  • Select the transit of Venus on 2012 June 6.
  • Under the button Multisite predictions using, select a site file. Then click the button. A form will appear with local predictions for all sites in the site file. You can generate predictions using any site file. In particular, you can create your own site files - using the Maintenance tab, Edit Site files.

  • Next to Transits during the century, change the list of transits to be for the century 1900, using the up/down buttons.
  • Select the transit of 1999 Nov 15. Note the blue line on the Earth maps, and the coordinates of that line displayed in the box at bottom left. This is a line separating total transit from partial transit.
  • Select the transit of 1937 May 11. Note the red line on the Earth maps, and the coordinates of that line displayed in the box at bottom left. This is a line separating partial transit from no transit.
