Statistic of observations; Lists of observers; Find events by star

When an 'interesting' occultation observation is made, it can be very useful to identify all previous occultations of that star. It can also be useful to identify previous occultations of a star prior to an observation. However these tasks have not been readily available for asteroidal occultations,

This form provides the following functions:

1. List all observers for the specified year range

Set the year range, and click List All observers for the Year range, The output can be sorted either by the observer's family name, or by their number of observations

3. Identify all previous occultations involving a specified star

This function allows the identification of any observed asteroidal occultation events on the basis of the star's coordinates. It will also identify whether lunar occultations of the star have been reported, and whether there are any light curves available.

The coordinates can be entered manually in the boxes provided. More commonly, the user will specify the star by its catalogue number using the catalogue drop-down box, and the fields for the star number. However the ability to use the catalogue identification depends upon having the relevant catalogue installed in Occult.

Once the coordinates have been specified, click 'Find events for this star. The form will display:

  • all asteroidal occultation events involving that star
  • if the star is in the XZ80 catalogue (used for lunar occultation predictions)
    • the XZ catalogue number(s)
    • the number of lunar occultations of that star present in the Lunar Archive
    • the number of light curves available for that star.

8. RUWE distribution

Displays a bar chart of the number of events for RUWE values between 0.50 and 3.00. For RUWE greater than 1.40, the bars are brown.

The actual values used to plot the bar chart can be obtained using the menu item, with List... Copy. The graphic of the bar chart can be copied with List... Copy RUWE image. That menu option is only listed when the bar chart is displayed.
