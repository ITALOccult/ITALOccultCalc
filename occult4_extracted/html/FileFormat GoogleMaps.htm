These files are used to provide output to be used with GoogleMaps.

When creating the HTML file to use with GoogleMaps, Occult combines the FileHeader and FileFooter with Occult output to create the html file.

The GoogleMap_ServerDriver.js is Java code to be used on a server to display the predictions.
