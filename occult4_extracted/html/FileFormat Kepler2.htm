Kepler2 target star files

Occult maintains two files to support the Kepler2 mission. The format of those files is set out below.

Kepler2.dat

This file contains a full listing of the Kepler2 target stars, as established from time-to-time. Files containing the target stars for each field are downloaded from links given in each field description here.

Kepler2.dat is an ASCII file sorted by Right Ascension, with the following field definitions

Columns

Meaning

1 - 9

The EPIC ID number

10-11

The Kepler2 field number

13-22

Right Ascension (J2000) in degrees. Epoch is about 2014?

24-33

Declination (J2000) in degrees. Epoch is about 2014?

35-38

Magnitude (on Kepler2 system?)

40

Cadence. The Kepler 2 data is summed on-board at either 1 min or 30 min effective exposures. This field has two values: 1 =1 minute summations; 9 = 30 min summations

XZinKepler2.dat

This file contains a listing of XZ80Q stars contained in the Kepler2 target stars, as established from time-to-time. The file is created by matching the full Kepler2 data to the XZ80Q catalogue.

XZinKepler2.dat is an ASCII file sorted by XZ number, with the following field definitions

Columns

Meaning

1-6

XZ number

8

Cadence. The Kepler 2 data is summed on-board at either 1 min or 30 min effective exposures. This field has two values: 1 =1 minute summations; 9 = 30 min summations

10-18

The EPIC ID number

20-21

The Kepler2 field number

Notes

1. The Kepler2 field number is included so that when extra fields are added, stars previously matched to past observations can be identified via the field number.

2. The Ecliptic Plane Input Catalog [EPIC] can be downloaded from here.
