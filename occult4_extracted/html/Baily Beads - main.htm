Baily Bead identifications

Overview of Baily Beads - identification

This routine enables the identification of the lunar features that give rise to a Baily Bead during solar eclipses. Typically, Baily Beads will be recorded on a medium such as a video recording, together with a time signal. The times of individual events (bead forming/disappearing/breaking/merging) is deduced by replaying the video recording. However, the lunar features that caused each event are not easily identifiable - particularly since typically the camera is not in a well-defined orientation with respect to the lunar pole, and the details of the observed beads can be quite sensitive to site location.

The program provides an animation of a solar eclipse to facilitate bead identification. Input quantities are the site co-ordinates of the observer, and the date and time of the eclipse. The animation can be adjusted to more closely replicate the observed beads by applying a correction to the relative declinations of the Sun and Moon.

The animation can be saved as an animated GIF file. The animation parameters are set under Animated GIF.... Animated-GIF settings, and the file created using Animated GIF... Create animated-GIF. Guidance on the issues with creating an animated GIF file are given here.

The limb can be plotted at various scales. The plot can be incremented or decremented by small periods of time, to better identify the formation of a feature. Also, the plot can be set to give an animation over a period of time.

Features are identified by pointing with the mouse - the Watts Angle of the pointer being continuously displayed.

The Plot Scale controls the scale of the image of the eclipsed sun. If the scale is 0.4 or smaller, the image is of the whole sun. Otherwise it is an enlarged portion of the solar limb.

Form control

Baily beads analysis potentially requires viewing of several forms at the same time. To aid with this, each form has a menu item to force the form to 'Stay on top', so that other forms will not obscure the form. If the form has this status set, the menu item will change to Return to normal. Similarly the main form has a menu item Show... to display forms that have been closed.

Date and site details

To set the program to identify Baily Beads, you must first specify the date, and observer co-ordinates. This is done in the frame called Date & Site. The parameters to enter are:

  • Year, month day of the eclipse
  • Site longitude (measured +'ve to the East), latitude, datum, and altitude
  • Whether to draw as a mirror image [For optical systems that use mirrors, image reversal will occur. If there are an odd number of reflections, check the Draw as mirror image box, to reverse the plot to correspond to the viewed image. Where the image is viewed on a projection screen, the screen should be treated as a reflection. [Example: for a projection screen image produced using a Newtonian reflector, the reflections are (1) the primary mirror, (2) the secondary mirror, and (3) the projection screen - which is an odd number of reflections.]

Time details

The time of the plot is specified in the Time for the plot frame. Enter here the Hour, minute and second for the plot time (in UT).

The time of the plot can be changed by manually entering a new Universal time. However, to assist entering a new time, specify a Step size (secs) under Change Time. Then the time can be incremented or decremented by this amount by clicking the + or - buttons. The Step size can be any value, including decimal fractions of a second.

Plot Parameters

Under this frame the user can set a number of parameters to adjust the plot. The parameters are:

  • The Central Axis angle. The main plot as the axis angle specified here located at the top centre of the plot. Use this to orient the main plot so that the region of interest is centred - particularly important for large magnifications.
  • The Scale sets the size of the main plot. If the scale is set to 0.6 or less, the plot is of the whole image.
  • The relative Offset positions of the Sun and Moon can be adjusted in Right Ascension, and Declination. The offset is in arc secs., and is applied in the sense of position of moon from sun.
  • A Correction to the solar radius at unit distance can be applied. The correction, in arcsecs, is of the solar radius at mean distance.
  • The check-box Add profile correction specifies whether the corrections to the Watts datum determined by Appleby & Morrison are applied.

To draw the eclipse

Click Plot to see a plot of the eclipse.

The Librations and Moon-Sun offset frame at the bottom left displays;

  • The topocentric librations of the moon in longitude (l), latitude (b), and PA of lunar axis (C); and
  • The x,y co-ordinates of the centre of the Sun from the centre of the Moon, in arc secs (Y +'ve to true north)

The Cursor location frame at the bottom right displays the Axis angle, and height of the mouse cursor, relative to the lunar limb when the cursor is on the main plot..

When the cursor is on or near the displayed limb, a ToolTip text shows the Axis Angle at that point, and the height of the cursor above the lunar limb.

The eclipse can also be displayed as an Animationidh_6BBMovie - see below.

Drawing a detailed plot

The detailed plot of the lunar profile is drawn in a separate window which at start-up appears at the bottom of the screen (and always stays on top.) To draw a detailed profile, click at the relevant point on the limb of the main plot. A detailed plot centred on that point will now be drawn.

To adjust the width of the lunar limb covered by the plot, select the relevant width under the menu item Axis Angle range…. Widths between 4 and 180 degrees are available.

Dragging the border changes the size of the window.

If the Animation effect is enabled, or the time is incremented or decremented, the plot on the window is changed accordingly. However the location of the plot along the limb is not changed.

Displaying limb heights

The height of the lunar limb, the solar limb, and the difference between the heights, can be displayed as text. Position the cursor at the location of interest, and click the RIGHT mouse button. A form will display the heights for axis angles of +/- 1 degree of the cursor location, at intervals of 0.1 deg.

Recording events

Bead identifications are displayed in a small rectangular form, which also stays on top. When a site is selected, previous identifications are displayed in this form.

To record the identification of an event:

  • Select the time of the event
  • Display a detailed plot of the limb at the expected location of the event
  • Place the cursor on the point on the lunar limb that causes the Baily bead event that occurred at the plot time.
  • Click the LEFT mouse button.

On the Bead identification form, the plot time, Axis angle of the event, and the profile height (scaled to the apparent diameter of the moon, and incorporating the correction for the offset between centre of mass and figure) are entered at the end of the list on the form.

Events on the form can be sorted, either by time or by Axis angle, by selecting the appropriate sort under the menu item Sort/delete data.

An entry can be deleted from the list. Select the entry, then the menu item Delete... selected. You will be asked to confirm before the entry is deleted.

Data should be periodically saved using the menu item File... Save. If the program is exited, or a new site selected, while some data has not been saved, a prompt will occur.

Plotting recorded events

Any event in the list of recorded events can be plotted by double-clicking on the event. Both the main plot, and the detailed limb plot, will be centred on the axis angle of the selected event.

Analysing the events

Next to each event in the form that records the events is a check box. This is used to identify those events to be used to analyse the identifications to solve for the 3 basic parameters - the correction to the longitude, latitude and radius of the Sun, assuming the longitude, latitude and radius of the moon are correct. The solution is performed by a least-squares fit to computed residuals. In computing the residuals, the program derives the lunar profile height using the Axis Angle identified against the event. (Note: it does not use the height as recorded in the form). Corrections to the profile are applied if the relevant check-box in the Plot parameters box is checked.

To compute the residuals, click the menu item Solve at the top of the form that records the data. The results of the analysis are displayed at the top of that form.

The status of the check boxes is saved whenever the recorded observations are saved. When that site is re-loaded at a future session, the program will compute the residuals at that time.

To help with the identification of beads, it is suggested that several (at least 3) unambiguous events be initially identified. Record these events, and solve for the residuals. Then enter these residuals in the relevant fields in the Plot parameters box, so that subsequent plots are aligned closer to the observations. Repeat this process as more events are reliably identified.

Saving / retrieving site details and observations

To save the observations and site details

  • Enter the following items of data:
    • Year, month day and hour of the eclipse
    • The site name
  • Preferably, also enter:
    • Site longitude, latitude, datum and altitude
    • Mirror image setting
    • Central Axis angle
    • Magnification
    • RA and Declination offsets
    • Correction to the solar radius
    • Whether the profile correction is to be applied.

The details are stored in the following way:

  • a subdirectory is created in the directory \occult\Observations called Eclipse yyyy mmm dd, where yyyy mmm dd is the date of the eclipse.
  • within that directory, a file is created having the Site name as the file name. If the file exists, you will be asked to confirm whether you want to overwrite the file.

Note. The file contains all the site details and all observations.

To load a file:

  • Select the menu item File... Open
  • Select the directory corresponding to the date of the eclipse
  • Select the relevant site from the list of files in that directory.

Animation

To view an animation of Baily Beads during an eclipse

  • In the animation box, check the Animation On check box
  • Set the Central Axis Angle to a value that corresponds to the region of the moon where the 2nd or 3rd contact occurs. If in doubt, set the magnification to a low value (e.g. 0.5), plot, and use the cursor to find the appropriate watts angle.
  • Set the time for the start of the animation, in the UTC fields.
  • Set the time interval between each plot (in secs), in the Step size field under Animation. Decimal fractions of a second are allowed.
  • Set the duration of the animation (in secs), in the Duration field. Note. When this field is non-zero, the button labeled Plot is re-labeled to Start Movie
  • Click Start Movie to start the animation.

During the animation:

  • The seconds of the current display, reference to the Hour and Minute listed in the Universal time fields, is displayed in the button labeled Stop. Clicking that button will stop the animation.
  • Clicking the Pause button will halt the animation. The animation can be resumed where it was paused from, by clicking the Resume button. Or it can be resumed 10 steps previous, by clicking the Resume 10 steps back button. While the parameters on the form may be modified during a pause, the changes will only have effect when the animation is re-run. The changes will not occur when the animation is resumed.
  • If the detailed window is displayed, the location of the solar limb on that display is updated.

In the animation box is a disabled check box for Auto save. This is not yet functional. The intention is to add the capability to save the graphic output as an animated GIF.
