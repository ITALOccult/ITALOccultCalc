Watts.bin

The file Watts.bin contains the heights of the Watts charts at 0.2 degree intervals of L, B, and WA. The file is derived from the dataset archived at Vizier

The limb height is stored as a single byte, ordered as follows:

  • for each combination of L and B, there are 1800 height values (from Watts angle 0.0 to 359.8)
  • for each value of L, there are 81 sets of height values, corresponding to B= -8.0 to +8.0, at 0.2 deg intervals
  • There are 91 sets of Latitude groups, corresponding to L=-9 to L=+9 at 0.2 deg intervals.

The zero-based record number is obtained from: 

position = 145800 * Convert.ToInt32(Math.Floor(45-5*L +.001)) + 1800 * Convert.ToInt32(Math.Floor(40 - 5*B +.001)) + Convert.ToInt32(Math.Floor(5 * WAngle + 0.1))

The height is decoded from the byte value as:

Height = 0.027 * (Height - 128)
