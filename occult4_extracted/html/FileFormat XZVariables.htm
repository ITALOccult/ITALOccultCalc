XZVariables

The file of variable stars was generated by matching stars in XZ80Q with the coordinates given in the General Catalogue of Variable Stars (4th Edition) as provided in machine readable form from the Sternberg State Astronomical Institute. [http://www.sai.msu.su/groups/cluster/gcvs/gcvs]. The file versions were those available at the end of May 2002. The files used were:

  • vol1.dat - Volume 1 of the GCVS [Constellations up to and including Crucis, with all coordinates revised to J2000]
  • iii.dat - The consolidated GCVS as at end of 2001, with a mixture of B1950 and J2000 coordinates. Used for constellations after Crucis
  • NSV.dat - New catalogue of Suspected Variable stars. B1950 coordinates. Stars indicated as "variability doubtful or erroneous to the compilers" were not extracted
  • NSVS.dat - New catalogue of Suspected Variable stars - Supplement. Both B1950 and J2000 coordinates for virtually all stars.

The data is stored in a file with a fixed record length file of 94 characters. The records are stored in order of XZ number. The record for any particular XZ number is best located using a binary search.

The content of each record is as follows:
ByteMeaning
0 - 6XZ Catalogue number
7 - 36Variable name - as taken from the AAVSO Variable Star index
37-47Variable type - taken directly from the GCVS
48-53Maximum magnitude
54-59Minimum magnitude, or magnitude range:
( = value is the amplitude of variability
< = brighter limit of minimum
60-62Photometric system of the magnitudes
63-76Julian date of maximum (or minimum for certain variable types)
77-91Period (days)
92-93CR/LF sequence
