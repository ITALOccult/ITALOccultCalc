Baily beads limb plot

This form is where you make the detailed identification of the limb point responsible for an event, and add the event to the list of events.

Stay on top. Click this menu item to ensure this form stays 'on top'. Click Normal to return to the usual priorities for viewing forms.

Recording an event

The steps to record an event are:

  • Set the time of the event in the main form
  • Set the approximate axis angle of the event in the main form
  • Click Plot to draw the limbs
  • On the detailed limb plot, identify the point on the lunar limb responsible for the event. This might be on the basis of the Watts charts, or from the plot of past observations.
  • Point the cursor to that location. Right click - and the event time, together with the axis angle and limb height at the location of the cursor, is added to the list of observations.

Plot options

The options on this plot are available from the menu bar, as follows:

Axis Angle range

Here set the angular extent of the plot of the lunar and solar limbs - between 4 and 180 deg. The plot is drawn with the smooth lunar limb being the horizontal axis.

Height

Set the extent of the vertical scale - between 2 and 20 arcsecs.

Lunar profile options

The lunar limb is generated using LOLA data points. This plots the altimeter data points from the LRO LOLA, leaving it up to the user's eye to identify the location of the limb.

    List limb heights

    For those that want to assess the numeric value of the limb heights from the Watts charts, place the cursor in the area of interest, and left-click. A form will appear listing the limb heights at 0.1 degree intervals for 1.1 degrees on either side of the location.
