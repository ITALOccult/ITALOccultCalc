Weather predictions

Occult provides two forms of weather predictions:

  • displays predictions of cloud cover and temperature for a specified location (including along a graze path).
  • accesses the Weacharts of the 7Timer system to provide regional maps of weather conditions for 16 days into the future. The full Weachart system is available here. The functionality provided by Occult is a subset of the weacharts that are of most interest to occultation observers.

Basis of the predictions

The data and charts of 7Timer are generated using the Global Forecast System (GFS) model of the National Centers for Environmental Prediction (NCEP). A description of the GFS model may be read here . Updates to the GFS model can be viewed here. Anyone interested in the basis of the weather predictions in Occult should read this documentation.

Local weather predictions

Local predictions of cloud cover and temperature for a three-day period is enabled at several locations. The 3-day limit ensures the predictions have a good likelihood of being correct - although users will have to make their own assessments based on their local knowledge and experience. For lunar grazing occultations, the functionality displays the cloud and temperature along the graze path - which may assist on deciding on observing locations.

Regional weather charts

Important

  • The Weacharts are generated at 7Timer every 6 hours - but it takes several hours for them to be generated. Occult accesses the latest complete set of Weacharts.
  • You should always check the label on the displayed map to confirm that it is for the expected date and time.
  • If a map is not available, you will be advised. You should try a different time setting. This should only occur for times before 'now', and for times more than 16 days in the future.

Selecting a chart

Select the map region

Weacharts are provided for the 6 populated continents, and the world as a whole. Select the region relevant to you. This setting will be retained between sessions. Changing the setting will result in the relevant map being downloaded.

Select the map type

Maps are provided for cloud (total, high, middle and low), atmpospheric extinction, temperature and wind. Changing the setting will result in the relevant map being downloaded.

Select the date and time

Select the date and time for the map using the list of radio buttons.. Changing the setting will result in the relevant map being downloaded.

Some background

Some background about the generation of these weather charts,
  • the numeric GFS data is generated every 6 hours by the NCEP. The relevant GFS model data set is indicated at the top right of the display, as the Starting point for prediction.
  • Ye Quanzhi (7Timer) uses the GFS data to generate the maps for his Weachart web page. It takes several hours for the complete set of maps used in WeaChart to be generated. Typically, the complete set of maps is not available for 4 to 8 hours after the date/time of the relevant GFS model. [Occult only accesses those maps that relate to cloud, temperature and wind.]
  • the display of the date/time selection on the form logically starts with "today's" date (in UT). The date of the GFS model is either the same, or the preceding, day (in UT).
  • the hours of the maps in the 12-hour map period are either 0 and 12, or 6 and 18, depending on the initial hour of the GFS model. Similarly, the end of the 3-hourly maps depends on the initial hour of the GFS model.
