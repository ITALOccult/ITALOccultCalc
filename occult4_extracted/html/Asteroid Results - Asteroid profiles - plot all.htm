Asteroid profiles - plot all

This form is used to create a set of image files for selected asteroid occultations. Note that while the form is enabled, you will not be able to access other functions in Occult.

Selecting the events

The selection of events is the same as the selection of events in the Observations editor. Except that all the events satisfying the criteria are presented in a list, rather than a drop-down box. That is, use the option buttons and drop down boxes to select a range of asteroids; those asteroids will be listed, and the output will be based on that list.

Quality limit

The routine only generates a graphic output for those events which have at least the specified quality value.

Selecting a value in the list has the effect of including all events which have that quality value, or a higher quality value. For example, selecting 'good' will result in the output of events that are marked as both Good, and Excellent.

The output plots vary according to the quality in the following manner:

  • Excellent - are plotted with no ellipse, and no observed chords
  • Good - plotted with an ellipse, but no observed chords
  • Poor, and Time Only - plotted with an ellipse, and showing the observed chords as lines.

Output directory, and file names

The program generates a file for each event. The file name is made up of the asteroid number, name, and event date. The form allows you to specify a directory to save the files. You should consider creating new directories as needed.

File format

This sets the file format used for the images. The NASA PDS archive prefers JPEG.

Set image size

The image size of the output files is selected form the 4 options in the dop-down box. Note that the file size varies approximately as follows:

Plot in B&W or color?

This choice is set in accordance with the currently set mode for plotting in B&W or color. To set this, draw a single event in the Observations Editor, setting the Colour/B&W choice as required. Then generate the present output.

Create the plots

If the Asteroid Observations Editor is open, you will be asked if it is OK to close it. If it is not closed, the routine will not proceed.

As the files are generated, the plot form will display the current event being plotted. However when this routine is running, the plot area is set by the image size setting, rather than the size of the plot form. As a result, the plot area may look different from usual.
