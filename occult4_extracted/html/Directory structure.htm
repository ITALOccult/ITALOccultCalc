The directory structure used in Occult, and the content of those directories, is:

DirectoryContent
Occult4The main directory. It contains the executable file (Occult.exe), a DLL file that does all the work (Utilities.dll), the Help file (Occult.chm) and two configuration files.
../Asteroid.
../Asteroid/ObservationsA directory for collecting observation reports of asteroidal occultations
../Asteroid/ResultsA directory for holding finalised results
../Asteroid/Results/MPC FilesThe files of results to submit to the Minor Planet Center are created here
../Asteroid/Results/PDS FilesThe files for archiving in the NASA Planetary Data System are created here
../AsteroidsA directory for holding animated GIF files showing asteroid rotation
../AutoGenerated AsteroidsLocation of the output files for autogenerated files generated from the Asteroid List & Display form
../AutoGenerated Asteroids/MJDThis holds files to control whether output files should be generated for a particular event - in particular, if the elements currently being used are more recent than those used for a previous output.
../AutoGenerated GrazesLocation for auto generated graze outputs. Empty directories bearing site names are created in this directory for holding intermediate files.
../AutoGenerated Grazes/_PredictionsOutput location for auto-generated graze predictions - which are stored in zip files named after the site and prediction period.
../AutoGenerated Grazes/GoogleMap files.
../AutoGenerated LunarLocation for auto generated lunar occultation outputs. Directories bearing site names are created in this directory, with the predictions being held in those files with names based on the prediction period.
../Downloaded FilesLocation for files downloaded by ftp or http from within Occult
../Generated FilesLocation for a range of files generated within Occult for internal use
../LightCurvesLocation for the consolidated file of light curves, and associated index files
../Lunar ArchiveDefault location for the creation of Archive-format lunar occultation observations. Used by Regional Coordinators
../ObservationsLocation for saving lunar occultation observations
../Observations/DoublesLocation where lunar occultation double star reports (including any attached image file) are saved. Also the location for files used for the analysis of lunar occultation double star reports to derive double star separation and PA.
../Observations/LightCurvesLocation for light curve report files, pending submission
../Observations/LightCurves/ReportedLocation for light curve report files after they have been reported. The files are automatically transferred here when they are reported.
../Observations/ResultsLocation for the files of reduced lunar occultations
../PredictionsDefault location for saving prediction files
../Predictions/AnimatedGIFLocation for saving the individual GIF files used to create an animated GIF
../Resource FilesLocation of the key data files for use by Occult. Most files are static, or rarely changed/updated
../Resource Files/GaiaLocation for Gaia catalogies
../Resource Files/MoonLocation for daily Lunar ephemeris files created by occult. [The files in this directory will be deleted whenever a new ephemeris basis is made available to Occult.]
../ShapeModelsLocation of the files for DAMIT and ISAM shape models
../SitesLocation of the Site files used by Occult
../UpdatesDirectory used for the files used to update the Occult program files. This directory will only contain files during the program update process. DO NOT WRITE ANY OTHER FILES TO THIS DIRECTORY. When the next update is performed, all files in this directory are written to the Occult 4 directory, and then deleted from this directory.TD>
