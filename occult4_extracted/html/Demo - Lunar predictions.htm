Demonstration of Lunar predictions

For this demonstration, three examples will be used based on an observer located in London, Washington, and Sydney. You should follow the example for the location closest to your own location

1. Lunar predictions for a location

  • Select the Lunar predictions tab, then click Predictions for single sites
  • If you have set your site location in the User Settings, the Starting at options in the middle of the form should be set as follows:
    • European observers: +12hrs
    • North and South American observers, -6 hrs
    • Asian observers: + 6hrs
    • Eastern Australian/New Zealand observers: 0hrs

    Set a site location for predictions:

  • In the Select site for predictions box:
    LocationAction
    Americas* In the top drop-down box, select USA Short.site
    * In the bottom drop-down box, select Washington DC
    Australasia* In the top drop-down box, select Australia.site
    * In the bottom drop-down box, select Sydney
    Europe* In the top drop-down box, select World Main.site
    * In the bottom drop-down box, select London, England


  • In the Star Catalogue box, have the XZ catalogue selected

  • In the Objects box, check Stars and Planets

  • Set the dates as follows
    LocationAction
    Americas2007 Mar 11
    Australasia2008 Apr 23
    Europe2007 Oct 06

    then click Occultations. A list of occultations will appear.

  • Place the cursor on the first event in the list. Right-click, and select MoonMap. Note the event on the Map. Note how if you select a different event, the MoonMap is automatically updated to that event.

  • On the MoonMap, Right-click, and select Plot 10 stars, starting at selected. Note the positions and times of the 10 events. Close the MoonMap.

  • Select the following star in the prediction
    LocationAction
    AmericasX136804
    Australasia2366
    Europe98966
    then right-click and select Star Details. Note the details, including the details of the double star.

  • Select the following star in the prediction - which will be used for the remaining examples
    LocationAction
    Americas2383 (= tau Sco)
    Australasia2366 (Antares)
    Europe1487 (=Regulus)
    then right-click and select World Map. Note the occultation path drawn on the map.

  • Using the same star, Right-click and select View in GoogleEarth. Zoom in and view the occultation region.

  • Using the same star, Right-click and select Multistar prediction. Note the prediction list for each of the sites contained in the site file.

  • Right-click, and select Relist Occultations. Select the star we have been using, Right-click and select Compute Graze. Note the graze prediction. Also note the list of sites near to the graze path, at the end of the prediction.

  • On the graze coordinate list, select a line, right-click, and select Graze Path Map. Note the local-region map showing the path.

  • On the graze coordinate list, select a line, right-click, and select Graze profile.

  • On the graze profile, move the cursor around, and note the time and height displayed at the cursor.

  • On the graze profile, select the menu option Observed graze and Occultation data, P,D[+/-0.5]. Note the observations plotted against the profile.

  • Select the menu option Observed graze and Occultation data, Include ordinary occultations. Note the increase of data.

  • Try the various options for plotting the observed data.

  • Select the menu option Observed graze and Occultation data, List grazes used. Note the form showing the list of grazes and number of events. Select a graze in that list and double-click on it. Note how events from that graze are highlighted. Double-click other grazes, to see how the highlighting changes.

  • On the graze profile, place the cursor on a data point. Note the observation details above the plot. Right-click, and work through the three menu options to highlight events

  • Close the graze profile. Right click on the graze path, and select Relist Occultations. Click the Grazes button. In the list of stars, select the star we have been using (Americas = 2383, Australia = 2366, Europe = 1487

  • At the left side set the following values
    LocationAction

    AustralasiaNorth Limit, Start Longitude = 100, End Longitude = 160, Step = 0.5
    EuropeNorth Limit, Start Longitude = -15, End Longitude = 50, Step = 0.5

    Note the long graze path output.

  • Place the cursor on locations near the start and end of the path, right click, and select Limiting Magnitude Table. Note how the limiting magnitude varies along the path. Also try plotting the profile at various point along the path, and note how it changes.

  • Place the cursor on a path point, right-click, select Graze Path Map. Extend the longitude and latitude ranges of the map to cover the length of the graze path and note how the map is drawn. Note the drop-down box that contains 'pre-set values' for the map plot. You can set these in the User Settings to match your favourite regions. Note the plot Sites drop down box at top right. Try changing the site file to see what happens. Also, change the box next to it, which controls the output of names.


2. Lunar predictions for an object

There are times when you would like to know when the next occultation of a particular object will occur. Occult has functionality to make that easy.

  • Select the Lunar predictions tab, then click Predictions of single Objects

  • Set Search From to 2007; Set to 2008;

  • In the box for star, from the drop-down box select Regulus. Click Find Star

  • In the box at the bottom of the form, note the list of events. Select any one, right-click, and select World Map. See the occultation region.

  • Place the cursor in the list of events. Right-click, and select Plot ALL listed events. View the world maps of all events.

  • On the main form in the Planets group, select Jupiter. Set the Search to be from 2012 to 2012. Click Find Planet. Select the 2012 July 15 event. Right-click to view the World map.
  • On the TaskBar find the LunarOccultationPrediction form, and open it. Note that the date is set to the relevant date. DO NOT ADJUST ANY OF THE DATE ITEMS ON THE FORM. All the details for the particular object are resident in memory ready for prediction using the functions on this form. For Planets, the data includes their main moons.
  • Select the site file WorldMain.site, and London as the site. Click Occultations. Note the predictions for Jupiter, and its four moons - and the line indicating the durations for each event.
  • Select any line, right-click, and select Compute Graze. Note how the graze prediction has two blocks. One for the object being fully obscured by the moon, the other for the moon just touching the object. Right-click on a line, and select Graze Map. Note how two lines are now drawn, corresponding to the two blocks

  • Return to the form for predictions of single objects. Set the search period to 2006 to 2012. In the Asteroid box, select 1 Ceres. Click Find Asteroid. See the list of events in the box - which you can view using the tools previously described.

  • Assume a magnitude 4 nova has just been discovered - and it is located at RA 7h 0m 0s, +24d 0' 0". Enter its coordinates, magnitude and a name, and click Find Object. See the list of dates for occultation. If you right-click an event and display a world map, the star's details will be in memory for local prediction. Go to the LunarOccultationPrediction form, and generate a prediction. The visibility will be assessed on the basis of the specified magnitude, and the name of the object will appear in the star identifier field.


3. Lunar predictions - Multiple objects for multiple sites

Note The functionality on this form is intended for regional coordinators. The functionality uses the Site files to control output.

The following is a brief illustration of what this functionality provides.

  • Select the Lunar predictions tab, then click Predictions of Multiple Objects for multiple sites

  • In box 1 Select Sites, select the USA Short site file.
  • In the list of cities, check a nominal number of cities - Chicago, Washington, Boston and New York
  • Set the Star catalogue to XZ<9, and Objects to Stars and Planets.
  • In box 5 Set date range - set the date range to be from 2008 Jan 1 to 2008 Apr 1.

    Ordinary Occultations

  • Click Compute Occultations. Some progress bars will appear. When the program has finished, use Windows Explorer to look at the content of the directory Occult 4/AutoGenerated Lunar. You will find a directory for each of those site names. In each directory will be a text file and a zip version of the text file. The file name is the prediction period; the content is lunar occultation predictions for that site and period. [The zip and text file outputs can be controlled in box 4 Set output options]

    Grazing Occultations

  • In box 3 Options for Grazes check the boxes Prediction text, Profile, Regional Map. Set the Plot size to medium. Set the Observed data to P,D[+/-0.5], and file format to PNG. Check the box to Hide Watts profile.
  • Now click Compute Grazes. Monitor the progress bars. When the program has finished, use Windows Explorer to look at the content of the directory Occult 4/AutoGenerated Grazes. You will find a directory for each of those site names - each of which is empty (they are used in the prediction process). The predictions will be found in the directory '_Predictions'. There are a series of Zip files with the file name being derived from the site name and prediction period.

  • Open the zip file for Washington.
  • Open the file 20080101 to 20080401 Graze Summary for Washington DC Summary.txt, and note the basis of the prediction and the list of grazes.

  • From the ZIP file, open the Regional Grazes - USA Short.png file, and the Regional Grazes - USA Short.txt files. The text file provides an index to the map. Note that the extent of the map is governed by the region covered by the site file, not just the selected sites.

  • From the Zip file, open the 20080213 Graze of 470 Washington DC.txt, and note the graze prediction. At the end of the graze prediction, note the 'Results of Observer scan', which includes Philadelphia (which was not a site selected for the prediction. The Observer scan considers ALL sites in the site file.

  • From the Zip file, open the 20080213 Graze of 470 Washington DC.png, and note the graze profile.


    Fixed region grazes

  • In the box for Fixed region grazes, set the year to 2008. The fixed regions are set in the User Settings, with 5 regions being predefined.
  • Click Compute fixed region grazes. This option takes a while to run. Monitor the displayed date.
  • When the program has finished, use Windows Explorer to look at the content of the directory Occult 4/AutoGenerated Grazes/World.
  • In the directory there are graphic files showing the occultation path of each occultation of a star brighter than mag 4.0, and any planet.
  • Also in the directory is a graphic file for each region (1 to 5) showing all grazes in that region. The legend for the graphic is provided in an associated .txt file of the same name.
