Windows11 issues

Issue 1

A user experienced an issue in Windows11 that affected the behavior of the print dialog in Occult. The user advised:

A while back, I wrote to you about how the Occult4 print dialog no longer worked as it used to. Instead, when printing from Occult4, I got a separate Windows print dialog box that always defaulted to my default print settings, so I got a printout in Portrait mode. This issue was specific to Occult4 and did not occur with other programs such as C2A. This has been quite annoying, as I use Occult4 Add-in from OW to print all my path maps for occultations.

Today, I did a bit of research and found that a recent Windows11 update changed the Registry so that old 32-bit programs no longer used the legacy program’s print dialog.

There is a workaround for this problem. It requires reverting the Registry setting to the Legacy Print Dialog Registry used before the update. This is the link to the page that explains the issue and how to reset the Registry. Paste it into your browser.

https://www.winhelponline.com/blog/restore-legacy-print-dialog-windows-11

Here is the exact entry to make the registry change from the Command Prompt window.

reg add "HKCU\Software\Microsoft\Print\UnifiedPrintDialog" /v "PreferLegacyPrintDialog" /d 1 /t REG_DWORD /f

I did this workaround and it worked for me. Now when I print from the Occult4 Add-in, I get the old print dialog from Occult4 and it automatically prints in Landscape mode as before.

 The spaces in the entry are critical. To avoid problems, do a copy and paste. Avoid manually keying it.

Note that for compatibility purposes, Occult is built to run on any processor - both 32 & 64 bit.
