Double stars observed in occultations

This provides a list of double stars recorded in an asteroid occultation

Examples

Main display

The information provided is:

  • The WDS 10-character identifier
  • The star number
  • Magnitudes of the star used as the primary, and the other star
  • The solution number. The number of possible solutions are 1, 2 and 4.
  • Separation and uncertainty, in milli-arcsecs. Where no ± value is given, the solution is not strongly determinative
  • PA and uncertainty, in degrees. Where no ± value is given, the solution is not strongly determinative
  • RUWE of the star. Provided to assess the degree of correspondence of RUWE and the star being a double star
  • The date of the observation
  • Asteroid number and name.
  • Publication information - whether it is in the WDS catalog, and the publication details in the Journal of Double Star Observations
  • The observers of that event. All observers are listed, irrespective of whether they recorded both, one, or other of the components, or had a complete miss observation
    • The Display menu options (below) varies what data is actually displayed

    WDS summary line

    This displays the data in the standard WDS catalogue format.

    The end of each line indicates when the star is in the WDS catalog, or has been published in the Journal Double Star Observations (JDSO)

    WDS submission line

    This displays the data in the format needed for direct submission to the WDS.

    The end of each line indicates when the star is in the WDS catalog, or has been published in the Journal Double Star Observations (JDSO)

    Unsolved doubles

    There may be stars listed as doubles, where the double star information has not been completed in the relevant event. Such stars are excluded from the normal listing - in all formats. If the Unsolved doubles option is checked, the stars that are unsolved (and only those stars) will be displayed. 'Obviously', the dopuble star solution for these stars need to be undertaken in the Observation Editor.

    Filter by observer name

    The name filter box allows observations by a particular observer to be listed. If there are any characters in this box, they will be used for any display of the data that is not a WDS Submission or Summary line.

    The filter applies a simple test of whether any of the observer names for an event contain the string of characters specified in the Name Filter box. It does not use wild cards. While you can enter names in upper/lower case, the search is conducted on a case-insensitive basis

    Displaying light curves

    When the display includes the 'Double star solutions' line, you can display all available light cirves for that event by

  • selecting an event line, and
  • either right-click, or double-click, on it.

    • Saving light curves as image files

    If you want to create image files for the light curves of an event, click the option button 'Save Images' on the menu bar, then re-display the light curves (with a right-click or double click). The file name will be the text of the header line in the light curve image, and the default directory will be an Occult subfolder LightCurvePlots. The user will have to confirm the file name and location as each file is saved.

    Users will need to manually manage the number of image files held in this subfolder.
