DWD file format

These files are for use as input files for some IOTA software. Most users will have no need for these files. OCCULT creates these files by converting Occelmnt files. When creating these files the user has the option of specifying the step interval for the output using the radio buttons. The default output is an interval of 1 day. But for fast-moving objects and (especially) planetary moons where the motion is seriously non-linear, intervals of 0.1 or 0.01 days can be selected.

The data contained in the DWDFILE file is:

LineContent
1Asteroid number, name, magnitude constant G, phase constant H, & diameter in km
2Star no, star RA, star Dec, Mv, Mp, and 3-letter catalogue identifier.
3Date (year, month, day, hr, min), minimum dist (earth radii), max duration, mag drop, solar elongation, lunar elongation, moon % illuminated.
4 to 9daily ephemeris for 6 days, giving:
JD, RA, Dec, dist from sun, dist from earth, mag, phase angle.
