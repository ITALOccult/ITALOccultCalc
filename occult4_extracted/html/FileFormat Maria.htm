Maria.dat

MARIA.DAT contains the outlines of the maria and some ray systems. The file is used to place maria on a simple map of the moon. The data was provided by Bob Bolster, VA. The data is in the form of a series of line segments. Each line of the file is comma-separated ASCII as:

    P , Long , Lat
where Long and Lat are the selenocentric longitude and latitude of a point; and
    P = 1 if the line segment starts at this point;
    P = 0 if the line segment is a continuation from the preceding point.
