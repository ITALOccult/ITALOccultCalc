XZ_ZC.dat

This file give a cross reference from an XZ number to ZC numbers.

Currently, this file is not used in Occult

The file is sorted by XZ number. Only XZ stars that are listed in the XZ catalogue are included. To find a ZC number for a specified XZ number, perform a binary search for the XZ number.

The file contains 3244 records, with a record length of 5 bytes. The first 3 bytes contain the XZ number - given by

    asc(byte1)+256*asc(byte2) )+65536*asc(byte3)

The last 2 bytes give the corresponding ZC number - as:

    byte4 +256*byte5
