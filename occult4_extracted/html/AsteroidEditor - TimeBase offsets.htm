Asteroid observations - time base offsets

This is a list of suggested time shifts - on the assumption that the occulting object is circular. For non-circular objects, this correction is absolutely misleading. It also assumes that the correction is caused by an error in the time base, and not (for example) an error in only one of the D and R times. In general, the corrections indicated by this option SHOULD NOT BE APPLIED. If a correction is to be applied, it must be done manually by entering the relevant value in the Time Shift box for the relevant event on the form for recording the observations.

The primary uses of this list are:

  • to identify one or two observations which are clearly discordant with other observations;
  • to estimate the magnitude of any error in a time base - so that further investigations can be made with the observer to identify a cause - and THEN make an appropriate correction.
