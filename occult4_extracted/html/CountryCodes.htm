List of standard 2-letter and  3-letter country codes and State codes

This table lists the internationally accepted 2-letter and 3-letter country codes, for use when specifying the State/Country of an observer.

For observers in Australia, Canada, and USA, it is acceptable to use the relevant 2- or 3-letter code for the state, rather than the 2- or 3-letter country code of the country.
