Shape model data

Two index files - one for DAMIT and one for ISAM - for shape models are held in the subdirectory ShapeModels. The files are .csv files, with the following field content

Field Label Content
1 Asteroid number integer
2 Model number string (although DAMIT model numbers are all integer)
3λ ecliptic longitude of axis of rotation, degrees
4β ecliptic longitude of axis of rotation, degrees
5 Period Rotation period, hours
6 YORP The coefficient of the YORP effect. Only available for a very small number of asteroids. The YORP correction to the rotational angle (in degrees) is calculated as YORP * (JD - JD0)2 / 2, where JD is the date of the event.
7 JD0 The epoch for rotation, and also for the YORP effect - Julian day number
8 Phi0 The rotational phase at JD0. Degrees. Usually zero.
9 Quality For recent DAMIT models a quality rating of the shape model
10 Version The current version of the shape model. Specified as a date
11 Comment Various general comments concerning the model.  Text
12 Created The date when this model was first created. Present for most DAMIT models. For ISAM models, set to be the same as Version
13 Modified The date (if any) when the current model was modified. Only present for some DAMIT models; not present for ISAM models.
14 DAMIT asteroid ID The DAMIT system uses an asteroid identification system that is independent of the asteroid's number. This field gives the DAMIT ID
