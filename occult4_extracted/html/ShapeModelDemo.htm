Shape model demonstration

This form is intended to demonstrate shape model functionality, including a 360° animation

Controls on the form

Select asteroid

This table lists all asteroids which have one or more shape models on you computer. Click an asteroid to list all available shape models (under  the table), with the first one being displayed

Select Model

This set of radio buttons lists all the available models for the selected asteroid - with the model identification next to it. Click a radio button, and that shape model is displayed.

Form Opacity

This slider  control makes the form part-transparent - so that is can be overlaid a plot of an asteroid (in particular, an image generated outside of Occult).

Date and time boxes

These are used to set the date and time for the plot. By default, the date and time is displayed as the current date UT, with the time at 12 hrs UT.

Display 1-degree animation

When this is checked, then when Plot is clicked, the shape model rotates once about its axis.

Background

The colour of the background (sky) can be selected here.

Projection

The default projection is against the sky plane. That is, how it looks to an observer on Earth.

The Earth Plane shows the features on the asteroid's face that faces the Earth, but the profile is of the shadow of the asteroid as projected onto the Earth's surface.

Include axes of rotation

Select whether the axes of rotation are plotted. Red is for the north pole

Plot

Draws the shape model. If -degree animation is selected, the model fully rotates about its axis .

Displayed quantities

  • Plot width in pixels. The default is 600. This can be changed between 300 and 600 using the check boxes at the bottom.
  • Volume equivalent diameter - as a fraction of the plot width
  • Surface-equivalent diameter - as a fraction of the plot width
  • Quality - a measure of the reliability of the shape model solution. Given for DAMIT models
  • Version - the version of the shape model - given for ISAM
  • Comment - any comments associated with the shape model
  • Created - the date the model was first created. DAMIT only.  Not present for all models
  • Modified - the date the model was improved. DAMIT only.  Not present for all models
  • DAMIT V2 and V3. V2 is a superseded format. Always use V3.
  • Update list of models. When new shape models arte downloaded, this updates the list of models. Only needed if the form is not closed.
  • Download all models - Downloads all DAMIT models
