Display asteroid observations by an Observer

This form is provided to assist observers with identifying their past observations. Also, to identify observations made by an observer when there is a need to edit their observations.

Apart from identifying observations by the observer's name, the form has three specific selection criteria - NTP for observations relying on NTP as the time source. Phone for using a telephone (mobile or fixed) as the time source, and Electronic for observations made using small aperture self-aligning telescopes with an electronic detector).

In the box at the top of the form, enter a string of characters contained in the observer's name (or perhaps location). Generally you should use a short string in preference to a long string - to minimize the consequences of data entry errors in the person's name. Similarly, select a string of characters in the family name - avoid first names (which may be converted to initials) and avoid using initials (which may or may not have a period after the initial). Note that the search is case insensitive.

The output provides a block of two or more lines for each event where the observer filed matches the search string.

  • The first line identifies the event - asteroid number and name, date [yyyy mm dd] and star number.
  • Subsequent lines in each event block lists each observer that matches the search string - giving the name, longitude, latitude and altitude.
  • At the start of each observer line there are 3 numbers:
    • Pos - a sequential count of events where the observer recorded one or more positive observations. If the observer rcorded more than one positive observation at that event, those subsequent lines are indicated with a '+'. All Miss observations by the observer are indicated with a '-'
    • Stn - a sequential count of the number of stations set up by the observer
    • ref - the observation reference number in the Event observation data held in Occult.

    In the above example, the last block shows an event where the observer recorded 2 positive events, and 2 Miss events (the 1st being numbered; the 2nd being indicated with the '+').

    The list can be copy, printed or saved using the File... menu item

    The list of unique sites can be plotted in Google Earth using a button at the top of the form. This can assist in identifying possible transcription errors in site coordinates.

    For observations made using the Electronic scope system, the listing includes five additional summary lists:

    • PART 1 Electronic observations by Observer name
    • PART 2 Electronic observers with 2 or more POSITIVE observations
    • PART 3 Electronic observers with 1 POSITIVE observation
    • PART 4 Electronic observers with more than 1 observation - positive or negative
    • SUMMARY

    Displaying sites in Google Earth

    There can be times where it is useful to plot all the site used by an observer in Google Earth. For example, such a plot can indicate inconsistencies in past site coordinates. It can also be used to quickly identify a previous site location near where one might want to travel to observe an event. To plot all past observing sites in Google Earth, simply click the menu item with List... Plot sites in GoogleEarth.

    The labels on the GoogleEarth plot have two components separated by a slash (/). The first component is the sequential number used in the displayed list of sites. The second number is the record number in the relevant past event - again, as used in the displayed list of sites.
