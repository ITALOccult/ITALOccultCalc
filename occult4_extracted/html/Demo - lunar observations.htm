Demonstration of Lunar observations

This demonstration will give you a basic overview of the functionality, using a grazing occultation held in the file of historical grazing occultations.

1. Retrieve observations of a graze

  • From the Lunar Observations tab, click the View/analyse historical grazes 1706 - now button.
  • In the fields at the top of the form set the year to 1975, and from the drop-down box select Spica. Click List.
  • Place the cursor on any line of the event. [Note: in this example, only one event is listed. Where more than one event is listed, you place the cursor on the line of the event of interest.] Right-click, and select Place event in Editor. The Lunar occultation editor will now open, and contain the observations of this event.

2. Overview of editor pages

  • At the top of the form there are 4 radio buttons. Check the Report radio button. The complete ILOC report form is displayed.

  • Click the Header radio button. This page is for editing the 5 lines that appear at the top of the report form. Buttons are provided at the right to save the current text on a line as a default value and to insert the default value into the line. [This enables you to retain your 'home' site details and add them into new reports.]

  • Click the Sites & Names radio button. The left half of the form is for adding/editing site details. The right half of the form is for adding/editing the names of observers and recorders.
  • In the panel for Sites, select a site in the middle of the list. Then try the following and note their effect
    • In the coordinate section, there are radio buttons for the data format. Click each of the formats.
    • Next to Altitude there are radio buttons to switch between metres and feet. Try changing the setting.
    • Note the various drop-down boxes, which list all the options for completing the ILOC report.
    • Note the edit buttons in the middle of the page. Click the move site Up button several times and note the movement of the site. Click the Sort by Number to return the site to its location. [Note use the Move buttons and the Sort buttons to order the sites; use the Renumber button to sequentially number the sites in their current order.
    • Note the Add as new site button - to create a new site using the data shown in the edit fields, and the Replace selected button - which updates the currently selected site with the data shown in the edit fields..
    • Note the Set selected as default, and Add default as new site. These allow you to retain your home site as a default value, and insert it in a new report.
  • Note the edit controls for the panel for Names of Observers..., which provide the same functionality as the edit controls for sites.

  • In the menu, select Sites... Display in GoogleEarth. Note how the sites are plotted on GoogleEarth - and how this can easily identify data entry errors in site coordinates. Comment. All sites were located on roads. In plotting the sites, it is assumed the coordinates are on the WGS84 system. The displacement evident in the plot is due to datum differences with old datums. These differences are not corrected in this plot - but are corrected in the reductions.

  • Click the Events radio button. This is the page for entering the observations. Wherever possible the form provides the ILOC options by radio buttons or drop-down boxes.
  • Select an event and note how the data is displayed. Click the Identify star button in the middle of the form to see how the form can identify the star. Note the sort options and how the events can be re-ordered. [And the renumber button to renumber the events sequentially]

3. Reduce and plot

  • Click the Reduce & Plot button. A table showing the reduced observations is displayed. Note the options for sorting the listing that are provided on the menu bar.
  • Select a particular event. Right click, and select Display event against profile. This option is available for ALL events.
  • Place the cursor on a point on the lunar limb. Press the left mouse button, and move the cursor. Not how a line is drawn, Move the cursor so that the line is tangential to the lunar limb. The ToolTip box displays the slope of the lunar limb relative to the horizontal - which is needed for analysing double star observations. Close the profile.
  • Since this report is a grazing occultation, click the Plot graze events against profile button. See all event plotted against the lunar profile.
  • Place the cursor over any data point and note the details of the observation appearing above the plot.
  • The plot includes many of the options available in a lunar graze profile prediction for plotting past observations, and highlighting related observations. Try the various options under Observed graze and occultation data. Note how a plot in the [P,D] reference frame has very blurred mountains, whereas the plot in AA, L, B] is much closer to the particular graze results.

4 Ordinary occultation results

  • Close all the forms that have been opened so that you are back to the main form for Lunar Occultation Observations.
  • Click the View/Analyse historical observations 1623 - now
  • Select the radio buttion Read a file
  • In the drop-down box, select RGO sites.
  • Highlight station 005 01 - observer B Dawson. At the top of the form, click the Create report for 005 01 button. Note how the observations are included in the editor. Do not close the report form while you perform the next ‘merge report’ steps.
  • Return to the site list. Select the site 005 02 - which is by the same observer. At the top of the form, click the Merge report for 005 02 button. Repeat this Merge report step for site 010 02.
  • In the editor, check the Sites and Names radio button. Note how the three sites are listed. Click Reduce and Plot, and view a reduction of all the observations. [This may take a little while...]

  • Close the editor and reductions forms. On the form for selecting observations, check the Search for star radio button.
  • In the drop-down box, select the star Regulus. Click the Read button. The form now lists all reported ordinary occultations of Regulus. Select the menu item with List..., Reduce all and save. At the bottom of the form you will see the results of the reduction of each event. When finished, view the file Occult 4\Observations\Results\ZC1487 Residuals.txt, which contains the occultation residuals for all occultations of this star. [The file name is based on the star number - in priority order ZC, SAO, and XZ].

  • Return to the form for selecting observations, with the Search for star radio button checked. From the drop-down box, select Antares. Check the box Double stars - highlight. Then click Read. Note how certain lines are highlighted - they contain an indication the event involved a double star, or a possible double star.

  • Return to the form for selecting observations, and check the Search for name radio button. In the Observer's name contains box specify Smith. In the Site Name contains box specify Canada. Then click Read. This will list all observations where the observer's name includes the characters 'SMITH' and the site name includes the characters 'CANADA'. Note that the search is not case sensitive.
