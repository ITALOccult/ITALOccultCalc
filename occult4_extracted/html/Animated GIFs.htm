Creating animated GIFs

Occult can save animated graphic displays as an animated GIF file. The following is guidance about how to obtain the best results.

 

Setting the animated-GIF parameters.

Animated-GIF files have two parameters that control how the animated-GIF is displayed

  1. The time delay between the display of images in the file. The units are 0.01 secs. For visually smooth displays the delay should be no more than 0.1 sec.
  2. the number of times the display repeats. 0 = continuous loop. 1 or more = number of repeats (note - the images are displayed one time more than the number of repeats).

These are set under the menu item .... Animated-GIF settings. Clicking this menu item will display a form where these two parameters can be adjusted. Make the changes, then close the form. The settings are retained for use when an animated-GIF file is next created.

 

Creating the animated-GIF file

1. The first step to creating an animated-GIF file is setting the plot parameters to give the desired animation. That is, adjust parameters such as the step interval, plot duration, and the form size, and perform an ordinary plot of the animation - until you are satisfied with its appearance.

2. Adjust the animated-GIF settings (for time delay, and number of repeats).

3. Click the menu item .... Create Animated-GIF. This will run through the animation [ saving a series of GIF files in the directory ...Occult4/Predictions/AnimatedGIF]. Once the animation has been completed, you will be asked to specify a file name for the output file. DO NOT save the file in the directory ...Occult4/Predictions/AnimatedGIF.

 

Suggestions

  • The file size is a combination of the number of images in the file, and the size of the individual GIF files. Where the animated GIF file is excessively large, either reduce the plot size, increase the plot interval, or reduce the duration of the plot.
  • Match the step interval to changes in the image. A step interval that is too small will result in an unnecessarily large file. A step interval that is too large will result in a jumpy animation.
  • The quality of the GIF image may be slightly worse than the displayed animation. As a general rule, get the animation display to be what you desire before creating the animated GIF.

 

Technical info

The creation of the animated GIF file proceeds on the following basis:

  1. The program deletes ALL GIF files from the ...Occult4/Predictions/AnimatedGIF directory.
  2. The program then runs through the animation, and at the end of each animation step the current image is saved as a GIF file in the directory ...Occult4/Predictions/AnimatedGIF. The files are numbered sequentially, starting at 10000.
  3. At the end of the animation, the user specifies a file name for the Animated GIF file. It then concatenates the GIF files, with the required headers, to create the animated GIF file.
  4. In testing, occasionally a GIF file wasn't deleted (exact cause not known.) If this occurs, the corresponding component file for a new animated GIF file will be skipped.
