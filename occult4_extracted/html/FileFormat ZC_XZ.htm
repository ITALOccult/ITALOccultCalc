ZC_XZ.dat

This file gives a cross reference from ZC numbers to XZ numbers. The file contains 3539 2-byte records corresponding to the ZC number. The corresponding XZ number is held as a two-byte signed integer in each record - that is, The XZ number is determined as

    ASC(byte1) + 256*ASC(byte2)
If there is no XZ record for a ZC number, the record returns a value of zero.

To get the record in the XZ80 catalogue of a ZC star, get the XZ number from the ZC-XZ.DAT file, then use the XZINDEX.DAT file to get the record number in the XZ80.DAT file.
