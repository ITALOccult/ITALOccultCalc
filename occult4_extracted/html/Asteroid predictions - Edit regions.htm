Edit regions

This form allows you to specify the coordinstaes of multiple regions to use in the filter for paths crossing regions.

The left box contains a list of regions, alphabetically sorted by the names you allocate to each region.

In the main editing box at the top right, you specify the name of the region. You should try to make sure you give meaningful names to the region label. Following that you specify the latitude and longiitude of the four corners of the quadrilateral - in degrees. There is no requirement for the quadrilateral to be rectangular - it can be any 4-cornered quadrilateral, such as:

However you should not make the quadrilateral too large - especially in longitude - as this may adversely affect the reliability of the filter by reason of corners of the rectangle being below the horizon.

 To assist with correctly setting the corners of the quadrilateral, you can plot the selected region in GoogleEarth using the relevant button. To vary the region, change the coodinates and re-plot - until you are satisfied with the region. Then click Add as New Region. This will add the region to the displayed list of regions.

You can edit a region by selecting a region in the list (which will display the details for that region), making changes to the details (including to the region name) and then clicking Replace selected region. If you have renamed the region, it will appaer in the list in its new alphabetically sorted location

You can delete a region by selecting the region and clicking Delete selected region. You will be prompted to confirm before it is deleted.

Saving the edited regions

The editing done on the form needs to be saved. [The region data is held in a file Occult 4\Resource Files\AsteroidRegions.txt] Click Save Regions to save all the edits. You will be prompted to do the Save if you try to leave the form without having saved any edits to the regions.
