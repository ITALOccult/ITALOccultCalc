Gaia double stars (more precisely, non-single stars)

This form displays details of stars which are in the Gaia non-single star files. The Source catalog is VizieR I/357: Gaia DR3 Part 3. Non-single-stars.

The Gaia double star data can be accessed via:

  • Asteroid predictions, for displayed event - from the world map - using the menu item Gaia double stars (nearby)
  • Asteroid reductions, for display event - from the Observations Editor, using the menu item List... Gaia double stars (nearby)
  • For a selected star - Maintenance tab, View catalogue details for selected stars, using the Gaia doubles button
  • Generally, Ephemerides tab, View double stars in Gaia DR3 non-single-star files button

There are 17 files in this catalog. Occult has used files 1 to 14. It has not incorporated spectroscopic binary systems (56808 entries), nor models for sources compatible with an Variability Induced Mover (vim) solution (nssvinfl) (870 entries). The files created for Occult are in the Occult 4\Resource Files\Gaia folder. They are GaiaDoubles.csv (which contains all the data, and can be opened in Excel.) and  GaiaDoubles.inx (an index file containing the byte location for the start of the first record at 1 min intervals in RA). The file contains 724,823 entries.

The functionality displays all stars in the Gaia files that are within 0.25o of a specified coordinate in Right Ascension and Declination. In part, this is to ensure the user has feedback that the functionality is working. Where a match is identified (on the basis of matching to within 3 arc-sec), the text ***   Probable match   *** appears before the relevant entry. At this time, Occult does not provide any further functionality (such as plotting an orbit).

The data displayed is different, depending on the orbital or astrometric model used for the solution. In a small number of instances, a star has more than one entry - corresponding to different models used.

Computing PA and separation using the Thiele-Innes constants A, B, F, G

1. Compute the Mean anomaly at the required date.
   M =  360/Period * (JD - 2457388.405023 - Epoch)

2. Compute the eccentric anomaly E by solving (commomly by iteration):
  M = E - ecc*sin(E)

3. Compute
   X = cos(E) - ecc
   Y = Sqrt(1-ecc2) * sin(E)

4. calculate the offsets in x and y (offsets in RA and declination) as
  x = A*X + F*Y
  y = B*X + G*Y

5. Separation is Sqrt( x*x + y*y)
   PA is Atan(x/y)
