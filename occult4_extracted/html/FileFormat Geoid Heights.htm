Geoid Heights.bin

This file contains the height of Mean Sea Level above the WGS84 spheroid. The height at each location (in metres) is held as a signed byte. The data is stored at one degree intervals in longitude, from 180d to +179 deg, for each degree of latitude from +90 to -90 - giving a total of 65160 heights.

A web site that maintains all available gravity datums is here, and a site for generating a table of heights above MSL is here. As at 2017, the data in the Occult file is derived from the eigen-6s4v2 model.
