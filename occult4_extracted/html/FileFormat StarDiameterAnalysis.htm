The file of data to be used in the star diameter analyser uses a tagged line format. The basic structure of the file is two initial lines, followed by a group of lines for each observer.

the tags for the initial two lines are:
TagData item
Star DiaThe stellar diameter in mas, the uncertainty in the value, and the number of light curves used
Num ObserversThe number of observers. There will be this many groups of observations in the file

For each observer there is a group of tagged data as follows:

TagData item
RecordThe sequential number of the observer record
ObserverObserver name
Frame rateThe number of frames/sec. It is 1 divided by the time interval between each data point
Display SecsThe number of seconds of the data to be displayed in the main plots
LimbDarken0 if no limb darkening to be applied. 1 for limb darkening to be applied
Is Partial0 if a full occultation, 1 if a partial occultation
SaturationThe level of saturation in teh video signal. 0 = no saturation. 1 = clipped at the 95% level, 2=clipped at 90% etc...
Motion DThe rate of motion of the star normal to the asteroid limb at D, in mas/sec
Motion RThe rate of motion of the star normal to the asteroid limb at R, in mas/sec
Uncert DThe uncertainty in the rate of motion of the star normal to the asteroid limb at D, 0-4
Uncert RThe uncertainty in the rate of motion of the star normal to the asteroid limb at R, 0-4.
Location DThe offset from the middle of the data to the middle of the plot, for D. Identifies the frame for the middle of the event.
Location RThe offset from the middle of the data to the middle of the plot, for R. Identifies the frame for the middle of the event.
Count DThe number of data points in the D light curve
Count RThe number of data points in the R light curve
Dia DThe deduced diameter from the D light curve
Dia RThe deduced diameter from the R light curve
Weight DThe weight allocated to the D diameter measurement (0-5)
Weight RThe weight allocated to the R diameter measurement (0-5)
Cal Top1DThe data point number for the start of the top calibration region for the top of the D curve
Cal Top2DThe data point number for the end of the top calibration region for the top of the D curve
Cal Top SDThe standard deviation of the data for the top of the D curve
Cal Bottom1DThe data point number for the start of the bottom calibration region for the bottom of the D curve
Cal Bottom2DThe data point number for the end of the bottom calibration region for the bottom of the D curve
Cal BottomD SDThe standard deviation of the data for the bottom of the D curve
Cal Top1RThe data point number for the start of the top calibration region for the top of the R curve
Cal Top2RThe data point number for the end of the top calibration region for the top of the R curve
Cal Top SRThe standard deviation of the data for the top of the R curve
Cal Bottom1RThe data point number for the start of the bottom calibration region for the bottom of the R curve
Cal Bottom2RThe data point number for the end of the bottom calibration region for the bottom of the R curve
Cal BottomR SRThe standard deviation of the data for the bottom of the R curve
Cal Meas1DThe data point number for the start of the measurement region for the D curve
Cal Meas2DThe data point number for the end of the measurement region for the D curve
Cal Meas1RThe data point number for the start of the measurement region for the R curve
Cal Meas2RThe data point number for the end of the measurement region for the R curve
Comment DFree text comment for the D event
Comment DFree text comment for the R event
Data DFor each point in the D light curve, the frame number from the csv file, and the height value
Data DFor each point in the R light curve, the frame number from the csv file, and the height value
