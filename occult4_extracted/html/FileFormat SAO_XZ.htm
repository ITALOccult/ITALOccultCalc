SAO_XZ.dat

This file give a cross reference from an SAO number to XZ numbers.

The file is sorted by SAO number. Only SAO stars that are listed in the XZ catalogue are included. To find an XZ number for a specified SAO number, perform a binary search for the SAO number.

The file contains 26735 records, with a record length of 5 bytes.

The first 3 bytes of each record contain the SAO number - given by

    65536*asc(byte1)+256*asc(byte2)+asc(byte3).

The last 2 bytes of each record give the corresponding XZ number - as:

    asc(byte4) + 256*asc(byte5).

To get the record in the XZ80 catalogue of an SAO star, obtain the XZ number from this file; then use the XZINDEX.DAT file to get the record number in the XZ80.DAT file.
