XZConfirmations.dat

This file maintains a record of confirmation observations of stars discovered during an occultation. The format is as follows:

Column Format Meaning
0 - 6 OCCnnnnIdentifier of the star in the XZDoubles Discoveries file
7 - 12 ... XZ number of the star.
14-21 yyyymmddthe date of the confirmation observation
23-46 ...Confirmation observer's name
48-70 ...Reference for the discovery observation.
[ON = Occultation Newsletter]
72-73 ...CR/LF sequence
