SAO_ZC.dat

The file is sorted by SAO number. Only SAO stars that are listed in the ZC catalogue are included. To find a ZC number for a specified SAO number, perform a binary search for the SAO number.

The file contains 3227 records, with a record length of 5 bytes.

The first 3 bytes of each record contain the SAO number - given by

    65536*asc(byte1)+256*asc(byte2)+asc(byte3).

The last 2 bytes of each record give the corresponding ZC number - as:

    asc(byte4) + 256*asc(byte5).
