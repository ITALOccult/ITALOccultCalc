File format - Earth Orientation Parameters

The Earth orientation parameters are held in two files - a historical file EOP_old.dat, and a current file EOP_present.dat.

The format of both files is identical, with the values being stored as binary. The file format is:
ParameterFormatMeaning
xsigned byte, units of 0.01"Displacement of the celestial Ephemeris Pole along the Greenwhich Meridian - positive towards Greenwhich
ysigned byte, units of 0.01"Displacement of the celestial Ephemeris Pole along the -90-deg Meridian - positive towards Canada
dUT1integer, units of 0.001sUT1 - UTC

The data provided in the EOP series is as follows:
PeriodData interval
1846 - 18890.1 years
1890 - 19610.05 years
1962 - now1 day

EOP_old.dat contains the data prior to 1962. EOP_present.dat contains the subsequent daily series.

Note: dUT1 is (by definition) zero before the introduction of UTC in 1962.


To first order, site coordinates are corrected for polar motion (before generating geocentric coordinates) as follows:

dLatitude = x.cos(longitude) - y.sin(longitude)

dLongitude = (x.sin(longitude) + y.cos(longitude)) * tan(latitude)
