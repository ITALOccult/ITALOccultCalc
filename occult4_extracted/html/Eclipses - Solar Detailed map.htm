Solar eclipse - detailed map

This option provides a map showing the eclipse path, for a location specified by the user. the plot region is controlled in several ways

  • Set the nominal map boundaries (longitude and latitude limits) at the top of the form, then click Draw map.
  • Place the cursor on a point on the map that you would like to be in the centre, and double-click.

The map is a conical projection, unless the region is very close to the pole. On the map, the line of central eclipse is drawn in purple; the limits of the umbral shadow are drawn in white; the penumbral limits, and curves of eclipse beginning or ending at maximum eclipse, sunrise or sunset are drawn in cyan.

You can plot cites on the map, by selecting a site file from the drop-down box at the top. You can include names with the cities by selecting the relevant plot criteria in the drop-down box next to the list of site files.

The map has a right-click menu that is almost the same as for the world map. The functions are:
OptionDescription

Set-up Baily Beads for animationThis opens up the Baily Beads module, and sets the basic parameters required to generate an animation for the location set by the cursor, and runs an animation. Note that this is intended for locations near to the central path. If less than half the diametrer of the sun is covered by the moon, the animation will not display. Details (see end)
Lunar profile - generate & SaveYou can generate and save the lunar limb for the current Mouse location. The file is saved in the Predictions subfolder, with a file name of MoonLimb_yyyyMMMddLongxxxLatyyy.txt. The first two lines of the file give details of the date/time/location, and librations. Following are 360 lines for each degree in Axis angle, with each line having 100 values of the topocentric limb height at 0.01° in Axis angle and a height resolution of 0.01"
View in Google EarthDisplays the eclipse path outline curves in GoogleEarth - where (for example) you can zoom into any location along the eclipse path to investigate possible observing locations.
