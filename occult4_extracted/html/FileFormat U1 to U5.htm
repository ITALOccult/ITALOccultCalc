U1.bin to U5.bin

The files U1 to U5 contain Chebychev polynomials for the Uranian moons 1 to 5. The polynomials were derived by Paris Observatory on a fit to observations made from 1954 to 2003. The polynomials are valid for the period 2006 Jan 1 to 2011 Dec 30 21h 48m 36s. The source files in ASCII can be obtained here, as files NOE-coefcheb1.res to NOE-coefcheb5.res

The parameters of the Chebychev series are:

  • Start date 2006 Jan 1.0
  • Block size 29.99875 days
  • #coefficients - 300 for each of x, y and z coords
All data is stored as doubles. There are 902 data items for each block, as follows:
  • Start date for block - as an offset from the start date of 2006 Jan 1.0
  • Block size (days)
  • 300 cooeficients for X
  • 300 cooeficients for Y
  • 300 cooeficients for Z
with this repeating for all 73 blocks.
