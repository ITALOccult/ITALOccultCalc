deltaT.dat, deltaTA.dat

The program maintains two files for the difference between UT and TDT

deltaT.dat contains values of deltaT (the difference between UT, and Terrestrial Dynamical Time) for each year, in seconds. The table covers years since -500. The period -500 to 1600 is covered on a 100-year interval. Thereafter it is an annual interval. The format is:

    YYYY [year]
    tt.tt [value of deltaT]
with other years following sequentially.

deltaTA.dat, contains the difference deltaAT between UTC and International Atomic Time (TAI). UTC is the usual broadcast time signal. In the periods between the application of leap seconds, there is a constant difference between UTC and TAI. The file also includes the value of dUT1 immediately following the introduction of a leap second. The format of the data held in this file is:

    YYYY [year]
    tt.tt [value of deltaAT]
    t.tt [Value of dUT1]
with other years following sequentially. The data is stored chronologically by year. The year (and fraction of a year) represents the date at which a leap second is inserted into UTC - with the time difference applying until the next entry. The year must have a decimal position, to indicate when the leap second is applied. Valid values are '.0', '.25', '.5', and '.75' - although to date only '.0' and '.5' have been used. The relevant value of deltaAT is found in the entry for a year immediately preceding the first year greater than the specified year.
