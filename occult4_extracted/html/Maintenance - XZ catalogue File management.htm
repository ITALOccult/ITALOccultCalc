XZ catalogue File management

There are two sections to this function. A user update section, and an Administrator update section.

User updates

From time to time the files of double star details and variable star details, can get updated. When this occurs, it is necessary to update the various versions of the XZ catalogue that Occult uses to generate lunar occultation predictions. The functionality here enables that to occur.

Updating double star files

When an updated Double star catalogue (XZDoubles.dat) is made available, it and any associated files (in particular, XZDoubles Discoveries.dat) should be placed in the Resource Files directory, replacing the existing files.

Then click the Re-index doubles button. It will remove all double star links from the main XZ80 file, and then create new links.

Finally, click the Create catalogue subsets.... button, to recreate the various subset catalogues so that they incorporate the new links.

Updating the variable star file

When an updated Variable star catalogue (XZVariables.dat) is made available, it should be placed in the Resource Files directory, replacing the existing file.

Then click the Re-index variables button. It will remove all variable star links from the main XZ80 file, and then create new links.

Finally, click the Create catalogue subsets.... button, to recreate the various subset catalogues so that they incorporate the new links.

Administrator Updates

The second part of the form allows for manual editing of the double and variable star catalogue files. These functions are provided so that a coordinator can incorporate the results of observations and new discoveries into these files and distribute updated files to users. This functionality is not meant for individual use.

Double stars

The function Edit XZ Doubles and Occultation discoveries opens a form for comprehensive editing and updating of the double star data using the WDS and Interferometric catalogues. This is described in detail here.

The function Update XZDoubles with latest orbits accesses the Sixth Catalogue of Orbits of Visual Binary Stars to retrieve the latest orbits of bimary stars. The downloaded catalogue may be found in the Downloaded Files directory, together with a subset limited to the ecliptic. The file will only be downloaded if the existing version of the file is at least 100 days old. Any unmatched entries will be listed in the file SixthOrbit_notMatched.txt in the Downloaded Files directory.

Variable stars

The function Create new file of variables uses the AAVSO Variable Star index file to create a new file of variable stars - XZVariables.dat. It assumes that you have downloaded the AAVSO file of variables from the Downloads page. The routine will take a small number of minutes to run, as it checks each star in the XZ80 catalogue for a match in the AAVSO Variable star index. The match criteria is an entry within 3.6" in both RA and Dec. - with the limiting factor being the precision of the coordinates of many of the variable stars. When this function has been completed, it is necessary to use the Re-Index Variables button to update the XZ80 file, and then to create the catalogue subsets.

The function Edit XZ Variables opens an editor for manually editing the file of variable stars. However the primary way of updating the file of variables is to create a new file from teh AAVSO file, rather than manualy editing an existing file.

It is assumed that anyone editing these files will be fully familiar with the data content, which is given at:
XZDoubles
XZDoubles Discoveries
XZConfirmations
XZVariables
