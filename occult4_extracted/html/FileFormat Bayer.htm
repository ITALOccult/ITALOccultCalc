Bayer.dat

This file contains the Right ascension, declination, Bayer letter, and Flamsteed number, of stars. The data is contained in a random-access file, with a record length of 10, sorted by right ascension. The Right Ascension and declination are on the equinox of J2000, and are stored as decimal degrees. The data is stored as:
Field Definition Data type
1 Right ascension Single precision
2 Declination Single precision
3 Bayer Byte
4 Flamsteed No Byte

The Bayer field specifies the relevant number in the Greek alphabet: 1=alpha, 24=omega.

Many stars have the same Bayer letter, with a numerical superscript. The value of that superscript is given in number field for the Flamsteed number.

If the value of the Bayer field is zero, the Flamsteed number of the star is given in the field for the Flamsteed number.
