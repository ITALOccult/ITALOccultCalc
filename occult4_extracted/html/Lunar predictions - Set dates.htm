Set UT dates for the predictions

Setting dates is one of the most fundamental issues with generating predictions. In Occult, there are two basic approaches to setting dates:

  • set a date range, to generate predictions covering a range of dates. Such predictions would be used to publish predictions, and to identify future events to observe (that is, forward planning).
  • generate predictions for a particular day. This might be for forward planning, or for observing events 'today'.

Note that many of the advanced features for occultation predictions are only available for predictions generated for a single day.


Setting the date

In box 4 you set the date range for the predictions. Particular issues are:

  • Whenever you change the Start date, the End date will be reset to be the same as the start date
  • the 'Year' button sets the start date to January first, and the end date to December 31 - for the year set in the Start date line.
  • the 'Month' button sets the start date to the first day of the month, and the end date to the 31 day of the month - for the year and month set in the Start date line. [If the month has less than 31 days, the actual end date of the predictions will be the 1st day of the following month; for February it will be March 2 or 3].
  • the 'Day' button sets the end date to being the same as the displayed start date.
  • the 'Today' button sets the start and end date to being the UT date corresponding to the date, time, and time zone in the computer's system clock. [As a result, the date inserted here can at times be the day before or after the local date.]


IMPORTANT - Setting the 'Starting at ' value

Occult generates predictions using a time window of 24 hours - identifying all geocentric conjunctions that occur in the window, and then generating local predictions. Inevitably, near the start and end of a 24hr period there are events that fall within the preceding/following day.

The natural observing period is night-time. Accordingly, to minimise issues with the 'end' of the 24hr computing period, the computing period needs to run from about mid-day to mid-day. The 'Starting at' column of radio buttons is provided for this purpose.

The setting on the prediction form is retained between sessions. It is initially set when you first installed Occult4 - provided you specified your site longitude in the first box on the User Settings page. [Note: It will be reset whenever you change the site longitude on the User Settings page.]
