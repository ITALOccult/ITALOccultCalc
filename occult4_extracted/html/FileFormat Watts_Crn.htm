Watts.crn

The file WATTS.CRN contains data for correcting the Watts charts for the corrections given in Morrison & Appleby MNRAS (1981) 196, 1013-1020. The data stored are the values indicated in Fig 3 of that paper (on page 1016), and are stored as two-byte signed integers. The order of storing the data is:

  • Quantities in order dR0, dX, dY, dR2
  • Within each quantity, the values are ordered by librations in longitude and latitude. The values are ordered firstly by l, in accordance with:
    L = -7.5, -4.5, -1.5, +1.5, +4.5, +7.5
  • Then, for each value of L, the data is ordered by B in accordance with
    B = -6.5, -3.5, -0.5, +2.5, +5.5
    (ie there are 6 values of L, each accompanied by 5 values of B, for each quantity).
