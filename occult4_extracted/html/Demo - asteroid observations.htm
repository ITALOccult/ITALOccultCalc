Demonstration - asteroid occultation observations

1. Summaries of results

  • From the Main menu, click the Asteroid Observations tab.
  • Click the Observed Diameters button to view a listing of all derived diameters.
  • Click the Observed chords button to list all asteroids for which one or more asteroidal occultations has been observed. SElecting an asteroid will result in a display of the individual observed chord lengths - which might be used to analyse the diamter and shape of the asteroid.
  • Click the Double star discoveries button to list all double star discoveries made in asteroid occultations.
  • Click the Derived asteroid positions button to view a list of astrometric positions derived from the observations.
  • Click the Events involving distant asteroids to list all such events.
  • Click observations by an observer. At the top of the form, enter a string of characters to search for, then click Search. Inspect the list of events found.
  • Click Generate Graphics files for Many events to open a form which enables you to select a range of asteroidal occultation events, and save the graphic of all those events. [This function is mainly for publication and archiving purposes.]

2. Edit/view observations of an occultation

  • On the main form, click the Add/Edit/Plot observed asteroidal occultations. This opens the observations editor.
  • At the top right of the form, note the Historical Observations box. Observations are selected using the large drop-down box. The other controls in the box change the list of observations which are included in the large drop-down box.
  • We will now view an occultation by Interamnia. Select the Name option, then the letter I. From the large drop-down box you will see several occultations by Interamnia. Select the event with 71 observations.
  • At the bottom of the form, the individual observations are displayed. As you change the selected event, you can see how the edit fields change.
  • Above the list of observation the solution for the asteroid size, and double star details (if any) are displayed. At the right, the text MPC60649 of 2007 9 26 indicates that the astrometric position has been reported to the Minor Planet Center.

  • In the menu, select with Observations... Plot sites in GoogleEarth. Note two groups of observers - one in Japan, and the other in Hawaii. Note how you can easily identify data entry errors in site coordinates.

  • In the menu bar, click Plot event. The results of the occultation are displayed.
  • The menu option Plot options contains many plot options. Try each to see the effect.

3. Light curve analysis

  • On the main form, click Star diameter analyser. This opens up a tool for analysing multiple occultation light curves to derive a diameter for the occulted star.
  • On the main form, click Light curve simulator. This opens up a tool for simulating the occultation light curve when the star has an appreciable diameter, and is more than about 10% of the diamter of the asteroid. The simulator involves three forms. One is an overall control form. The second is a form for drawing the shape of teh asteroid, and the apparent path of the star against the asteroid, The third form displays the light curve. THe 2nd and third form have variable transparency - so that they can be overlaid against the asteroid reduction plot to draw the asteroid shape, and to overly the light curve over the observed light curve to assess the goodness of the fit.
