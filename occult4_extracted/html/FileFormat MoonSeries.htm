MoonSeries.bin

This file contains a series used to compute the position of the moon for periods not covered by the DE405 or DE413 ephemeris. It is an extraction of the series of Chapront (ELP-82) which was fitted to DE200. [The full series is available on the CD-Rom Astronomical Catalogues Vol. 3, from the Astronomical Data Centre.] All terms with coefficients >= 0.0005” have been included. The data within the file has been stored as follows:

  • Main problem, large terms (Number of terms in Longitude, Latitude & Distance = 27, 14, 26)
    • D As Integer
    • Lp As Integer
    • l As Integer
    • f As Integer
    • Arg As Double
  • Main problem, small terms (422, 386, 332)
    • D As Integer
    • Lp As Integer
    • l As Integer
    • f As Integer
    • Arg As Single
  • Planetary perturbations [Neptune & D] (1398, 590, 1047 ; by Time 50, 8, 26 )
    Planetary perturbations [D & Lp] (30, 24, 27 ; by Time 65, 42, 57)
    • Mercury As Integer
    • Venus As Integer
    • Earth As Integer
    • Mars As Integer
    • Jupiter As Integer
    • Saturn As Integer
    • Uranus As Integer
    • Neptune_D As Integer
    • D_Lp As Integer
    • l As Integer
    • f As Integer
    • Phase As Single
    • Arg As Single
  • Earth figure (75, 64, 62; by Time 1, 1, 0)
    Tidal effects (1, 0, 0; by Time 1, 0, 2)
    Moon figure (1, 0, 1)
    Relativistic (2, 0, 2; by Time1, 0, 2)
    Solar eccentricity (by Time2 5, 0, 4)
    • Zeta As Integer
    • D As Integer

    • f As Integer
    • Phase As Single
    • Arg As Single
Corrections applied are:
  • To longitude: -0.09845” (difference in 'departure point' between dynamical ecliptic & FK6)
  • Light-time: computed using the 4 largest terms in distance.
