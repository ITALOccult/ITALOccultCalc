Moon_yyyy.bin

These files are labeled Moon_YYYY.bin, where YYYY are the digits of the year. The files are located in the directory ...\Resource Files\Moon. There is one file created for any year - a random access file of record length 74. The data is stored as 17 fields as Integers (giving the 6 polynomial coefficients in RA, 6 in Dec, then 5 in Parallax), and three short integer fields (giving any offset in longitude and latitude that has been applied to the ephemeris, and the basis of the ephemeris).

  • The values for RA have to be divided by 107 to get the coefficients in degrees, and then add 180.
  • The values for Declination have to be divided by 107 to get the coefficients in degrees
  • The values for parallax have to be divided by 108.
  • The values for the longitude correction and latitude correction have to be divided by 10,000 to get the correction in arcseconds.
  • The field giving the basis of the ephemeris has values of '405' [DE405], '413' [DE413], or '87' for Chapront [effectively, DE200]

Each record corresponding to a day in the year. The file will only contain data for dates computed by OCCULT, with other dates having null values. [If the data is not present for a date for which predictions are requested, the program will compute the ephemeris for that date.]
