Observed grazes

Details about the source of the graze data in Occult are given here.

List and analyse graze events

You can obtain full access to the observations made at grazing occultations. Grazes are identified by entering relevant search criteria in the following boxes at the top of the form.

The function will work with the old format files of graze observations used in Occult prior to version 4.0.4.1 - by checking the box 'use old format files'.

For the Date, you can specify any one or more of Year, month and day. For example, enter 28 in the Day, leaving year and month blank - and all grazes observed on the 28th day of any month will be returned. For the year, you can enter just some of the digits - eg '18' will retrieve all grazes in the 1800's, as well as for all years ending in 18, such as 1918.

For the star number, you can select the star from the drop-down box, or enter its number in any of the ZC, SAO or XZ boxes. For the drop-down box there are 6 sets of names available under the menu item Set 'Name of Star' list. Those sets are:

  • All names
  • Proper names
  • Bayer letters
  • Magnitude brighter than 2
  • Magnitude brighter than 3
  • Magnitude brighter than 4
    • To clear the star number fields, set the XZ number to 0.

    For the observer's name, enter a string for part of an observer's name. The string is preferably part of the family name. Avoid personal names, or initials. The search based on the name string is case insensitive.

    Click List Events to display all graze events that meet the search criteria.

    The graze events are displayed on the form. The display depends on whether the new or old format files are being accessed:

    • New FormatEach event is separated by a short line of asterisks. The data for each event is as follows:
      • The date of the first event in the graze (year, Month, Day);
      • Graze identifier - a uniqe identifier applied to each graze;
      • The star involved in the graze
      • The number of events associated with the graze;
      • The number of different observer names associated with the graze. {Note. For many grazes in the 1960's there are no observer names available. All missing names are treated as the same observer in this count.}
      • A list of distinct observer names. [Note - occassionally typographcal errors result in a name appearing more than once - with different spelling.]
    • Old Format Each event is separated by a short line of asterisks. The data for each event is presented like an ILOC occultation report - with the first section being Sites, the second section being observer names, and the third section being the reported events.

    Analysing an event

    When an event of interest has been located, you can analyse it in full detail by transferring the observations to the Lunar Occultation Observations Editor, by placing the cursor anywhere on the particular graze results, Right-click, and select Place event in Editor. This will open up the Observations editor, loaded with the graze results.

    Having placed one of those reports into the Observations Editor, you can merge other reports with it by placing the cursor anywhere on the further graze result, Right click, and select Merge event with event in Editor.

    With the observation in the editor, you can reduce the observations and plot the results - including plotting the results against all other occultation observations against that part of the lunar limb.
