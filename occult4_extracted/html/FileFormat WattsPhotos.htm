WattsPhotos.dat

This file gives the details of the photographs used to generate the Watts charts. The file is an ASCII file with 533 entries. The first line is a header line. In each line, the data are:
BytesData item
0-2Sequential order number of item in the Watts book
3-8Plate number
9-28Date of the photograph.
29-34L libration
35-40B libration
41-44Start of arc
45-48End of arc
