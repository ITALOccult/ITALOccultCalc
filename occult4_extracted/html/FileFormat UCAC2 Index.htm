UCAC2 Index.dat

The Tycho2 catalogue is structured in half-degree bands. UCAC2 Index is a random access file with record length 12, containing the number of the first 2UCAC star in each band of the catalogue, and the first and last records in the Tycho2 file that need to be scanned to find the 2UCAC star. The record structure is:

0 - 3number of the first 2UCAC star in a declination zone (integer)
4 - 7first record (unary based) in Tycho2.bin for the scan
8 -11last record (unary based) in Tycho2.bin for the scan
