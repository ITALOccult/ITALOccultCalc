Light curve format

A light curve file has 5 logical lines, each terminated with a CR LF sequence. Each line has data elements separated with a colon (:). The first element in each field is an identifier.

F5.2UT Sec - for first data point
F6.2Time difference between the first and last data points (secs)
I5Number of data points in the light curve

ValuesLight curve values, as integers. This line is a single line of indefinite length. Importantly, the line must not include any line breaks. [In the examples below, the lines are wrapped for the purpose of display only].

The values have been scaled such that the maximum value is set to 9524 (=10,000/1.05). As a result, all light curve values have a maximum near to, but less than, 10000 - irrespective of the numeric values in the source light curve. (No adjustments to the zero-level in the light curve data are applied.)

Where the light curve is missing one or more exposures, the data element for that exposure is empty.

The number of elements is specified in the last data element under Date.

The consolidated file of light curves LightCurves.txt is simply a concatenation of multiple individual flight curve files, sorted by date.
