Light Curve Embargoes

This form enables the setting of embargoes for submitted light curves.

When an embargo is set, it is applied to pending submitted light curves. It does not apply to light curves that are already added to the main file.

When a file has been embargoed, the file will have an added extension of '.embargoed'.

To set an embargo, you:

  • click the header line in the list of Current embargoes. This will clear the data entry fields, and set the default start and end dates.
  • specify the asteroid number
  • set the year and month of the start of the embargo period. The default displayed start is 6 months before 'now'.
  • set the year and month of the end of the embargo period. The default displayed end is 18 months after 'now'.
  • specify the name of the person who has requested the embargo
  • specify the contact details for that person
  • click the button "add as new Embargo

 The details of an embargo are displayed in the listing of Current Embargoes.

The details of an existing embargo can be edited by selecting it in the list of Current embargoes - which will display the details in the data entry fields. Change those fields as required. Then click Replace selected embargo

An embargo can be deleted by selecting the embargo, and clicking Delete selected embargo

The current embargoes are saved whenever there is a change/addition/deletion of embargoes

The list Embargoed files is a list of all light curve files that are marked as embargoed on the basis of the displayed list of embargoes. This list is updated whenever there is any change in the embargoes.

When the main form menu option to display all submitted light curves is selected, all embargoed light curves will appear at the end.

An embargoed light curve can be edited at the same time as any other light curve.

A light curve marked as embargoed will not be added to the main file. It will remain in the LightCurveReports folder until the embargo expires.
