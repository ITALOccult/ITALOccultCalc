Baily bead results

This form records the identifications of the beads.

To select an item, place the cursor on the item & click.

To check an item, first select it, then click the item again.

To plot the limb showing that event, select the item - then double-click. [In the process, the check box status will change, and then return to the previous state.]

Events are entered into this form by right-clicking on the plot of the lunar limb at the point of the lunar limb causing the event details.

Events can be sorted by time, and by axis angle - under the Sort menu item. This allows easy location of a particular event.

You can delete a selected item, or all items, using the Delete menu item.

You can do a solution based on all the checked items. The solution computes the correction to the relative RA and declination of the sun and moon (displayed as a correction to the position of the sun), and to the radius of the sun - using the Solve menu item.

Stay on top Click this menu item to ensure this form stays 'on top'. Click Normal to return to the usual priorities for viewing forms.
