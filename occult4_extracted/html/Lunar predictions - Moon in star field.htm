Display of Moon in star field

This option displays the apparent position of the moon against the background of the stars in the XZ80Q catalogue.

The form opens showing the current UT date and time, and the site location is that set in the User settings. Adjust these as required.

For an animation, specify the duration to be covered by the animation (in minutes) and the step interval to be used in the animation (in secs)

The plot can display 4 different fields of view - 0.7, 1, 2, and 4 degrees. Select the relevant field size. The larger the field size, the slower the animation.

The animation can be saved as an animated GIF file. The animation parameters are set under Animated-GIF.... Animated-GIF settings, and the file created using Animated-GIF... Create animated-GIF. Guidance on the issues with creating an animated GIF file are given here.

The map drawn on screen has the following background colour coding:
ColourMeaning
Blacknight-time
Various shades of bluetwilight
Light greysun above the horizon
Brownmoon below the horizon

If an animation is running, it can be cancelled by clicking the Cancel menu item that appears during an animation.

The time of the current display is indicated below the hr and min fields.

Suggestion: One of the things that can be illustrated in the animation is the changing direction of motion due to diurnal parallax. To see this effect, set a duration of 5000 minutes, and a plot interval of 300 seconds. During the animation watch the direction of movement of the stars. If there was no parallax, the directions would change in a continuous manner. However because of the effects of parallax, you will see the directions of motion changing in an oscillating manner.
