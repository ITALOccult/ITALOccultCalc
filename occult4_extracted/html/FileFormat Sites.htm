Site files

The site files are used to control the output generated by Occult.

BytesMeaning Comments
0-10 Longitude Decimal degrees
11-21 Latitude Decimal degrees
22-26 Altitude Meters
27-31 Telescope aperture cm. Used in lunar occultations to compute magnitude limit.
32-36 Magnitude correction Used in lunar occultation predictions, to vary the computed magnitude limit.
37-42 Lunar Graze travel distance km - for stars fainter than 6.5
43-74 Site name Text
75-76 Flag to control plotting on maps values 0 - 4. Used to control how sites are plotted on maps. 0=never, 1= only on detailed maps, 2= only on medium maps, 3= on all maps, 4=mobile site [not used]
78-86 Site short name (for use on maps) Text
87-92 Time zone Hours
93-94 Flag to control plotting on GoogleEarth values 0 - 2. 0=never, 1=use low precision (coordinates rounded to nearest 5 minute value - so that actual location not shown), 2=use full precision.
