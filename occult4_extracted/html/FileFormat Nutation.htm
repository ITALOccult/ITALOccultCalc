Nutation.bin

This file holds the series for computing nutation. There are 63 terms included in the series. The data is stored as a random access file with record length 32. The field definition for each record is:

    D As Integer
    m As Integer
    MP As Integer
    f As Integer
    OM As Integer
    NutL As Double
    NutLt As Single
    NutB As Double
    NutBt As Integer

Lt and Bt are terms multiplied by time.
