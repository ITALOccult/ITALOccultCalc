Miriade solutions

The details of the solutions available in the Miriade system are available by clicking the Get solution details button on the Asteroid Search form. The form that appears displays the details like:

 

The small list box  at the left displays the list of asteroids currently available in Miriade. The solution details are displayed by simply changing the selected asteroid (with the selection process resulting in a query being sent to Miriade).

The columns that are displayed have the following meanings:

Sn The solution number. A solution gives the best fit of the components to their observed relative positions
System Number The asteroid number
System Name The name of the asteroidal system - which corresponds to its 'traditional' name
Component Name

The name of the component in the system. For the main body in the system, this will be identical to the Name. Otherwise it will be :
* the proper name of the satellite (if it has been named);
* an identifier such as Antiope A and Antiope B {for (90) Antiope} if the main components are of a similar size; or
* the discovery designation [which is in the form Syyyy-nnnnn-x, where yyyy is the year of discovery, nnnn is the asteroid's number, and x is the sequential number of the discovery in that year.]
Solution method The method used to generate the orbit. Kepler = fitting Keplerian orbits to the observations. NumInt = a numeric integration technique
Fit mas Accuracy of the orbital solution, in mas
Prob % Probability of the orbital solution being correct, between 0 to 100 %
Radius  (km), ±km The radius of the component, and its uncertainty, in km. This radius can be different as between the solutions
Solution date & Time The date and time the orbit solution was generated/entered into Miriade
Eph y = there is an ephemeris; n = no ephemeris
Ephemeris range - start The start date of the available ephemeris
Ephemeris range - end The end date of the available ephemeris
Err y = there are error values associated with the ephemeris; n = no error values
Error available start First date for when error values are given in Miriade
Error available end Last date for when error values are given in Miriade
