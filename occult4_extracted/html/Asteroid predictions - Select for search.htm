Select asteroids for search

This form can open in two styles

  • The user can select either a single asteroid, or a range of asteroids; or
  • Only a single asteroid can be selected.

To select a single asteroid, either click the asteroid followed by OK, or double-click on the asteroid. Click the form close icon at the top right to close and return without any asteroid being selected (assuming none have been highlighted).

To select a range of asteroids - use the mouse to highlight the selected range in the usual way (i.e. click & drag; or click, move to another location, hold Shift, and click again) Click the form close icon at the top right to close and return without any asteroid being selected (assuming none have been highlighted).
