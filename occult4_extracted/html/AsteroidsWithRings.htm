Asteroid Rings

As at 2014, 1 asteroid (Chariklo) is known to have a ring system.

Edit file of asteroid rings

The file of asteroid ring systems can be edited from the Maintenance tab. The data to be entered for each asteroid ring system is
  • The number of the asteroid (un-numbered asteroids are not catered for)
  • the Right Ascension and Declination of the pole of the ring orbit (J2000, in degrees)
  • for up to 5 rings, the radius of the ring, in km.

The file also contains a field giving the number of rings in the system. This is calculated on the basis of the non-zero ring radii.
