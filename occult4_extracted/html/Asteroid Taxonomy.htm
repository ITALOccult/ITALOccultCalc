AsteroidClasses.csv

This file contains the taxonomic classification of asteroids. This classification can be used to filter searches and observations, and is displayed on predictions.

NOTE: The file only contains numbered asteroids. That is, Occult does not provide taxonomy for unnumbered asteroids.

The file in Occult of taxonomic classes is created from the lists at the Minor Planet Center web site, using the button on the Maintenance tab (under Maintain certain data files) Download and create file of Asteroid classes. This would generally be an administrator-type process, with the resulting file uploaded to the IOTA web site for downloading via the usual General downloads button (under item #9). However users can generate the file themselves if they so wish.

The file AsteroidClasses.csv has the following structure:

Asteroid number Class

The possible values for the classes are:

Class Characteristic (from the Minor Planet Center)
Amor Amors have perihelion distances between 1.0 and 1.3 AU
Apollo Apollos have perihelion distances less than 1.0 AU and semimajor axes greater than 1.0 AU
Aten Atens have semimajor axes less than 1.0 AU
Centaur Centaur objects have perihelia beyond the orbit of Jupiter and semimajor axes inside the orbit of Neptune
Trojan Asteroids sharing an orbit with a planet, but which are located at the leading (L4) and trailing (L5) Lagrangian points (that is, 60° ahead or behind). Most Trojans are Jupiter Trojans (about 4800 as at 2018), but there are also 5 numbered Trojan asteroids for Mars, and two for Neptune.
TNO Transneptunian objects (TNOs) have orbits with semimajor axes beyond the orbit of Neptune. Some of these objects, generally assumed to be in librations with Neptune, have perihelion distances inside the orbit of Neptune
PHA Potentially Hazardous Asteroids. Objects with H brighter than or equal to V = 22.0 (equivalent to a diameter >100m) and an Earth MOID (Minimum Orbit Intersection Distance) less than 0.05 AU. Look here for an explanation of MOID

 

The attribute PHA is always additional to another attribute (Amor, Apollo or Aten), and is listed like Apollo + PHA
