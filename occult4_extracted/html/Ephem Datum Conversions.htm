Datum Conversions

Local datum to WGS84

The latitude and longitude referenced to 'old' local datums can be converted to the WGS84 datum using this utility. The basis of the conversion is the Molodensky parameters given in the authorative WGS84 reference documentation TR8350.2.

Note that the Molodenski conversion parameters give a conversion accurate to about 0.1".

The drop-down box provides the list of datums included in this conversion. They correspond to the datums used by ILOC in association with the dataset of lunar occultation observations - with the number being that allocated by ILOC. The list includes a number of modern datums that are formally identical to, or practically identical to, WGS84 - and these are indicated with '(= WGS84)'.

TR8350.2 includes Molodensky conversion parameters for regions within datums. However all conversions done within Occult are based on the conversion parameters given for the entire datum.


Mean Sea Level (msl) above WGS84 spheroid.

The reference used for all GPS measurements is the WGS84 spheroid.

The mapping authorities around the world have adopted the WGS spheroid (or very close national equivalents) as the reference point for measurements of latitude and longitude.

However, the mapping authorities around the world have also retained msl as the reference surface for measuring altitude.

The default display of almost all GPS units is WGS84 latitude and longitude, and msl height. Internally, the GPS unit applies a correction to a computed height above the WGS84 spheroid to obtain the msl altitude. Some GPS units have an option to display the height above the WGS84 spheroid.

Occult maintains a file to convert from msl to spheroid height - which is required for accurate analysis of observations. This option displays the height of the mean sea level datum above the WGS84 spheroid for any longitude/latitude location. A web site that maintains all available gravity datums is here, and a site for generating a table of heights above MSL is here. As at 2017, the data in the Occult file is derived from the eigen-6s4v2 model.

Note When reporting observations, report height with reference to the msl datum. If you report a WGS84 referenced height, it is very important that you indicate this in the report.
