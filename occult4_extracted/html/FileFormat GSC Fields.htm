GSC Fields.dat

GSC Fields.dat contains the RA and Dec limits for each of the Guide Star Catalogue fields. It is used, together with a Gaia...inx file, to identify a Tycho2 star from its number.

This file is present in Occult, being a file in the Occult installation files.

The file is a random access file of length 16. There are 9537 records, each corresponding to the GSC region number (i.e. the first 4 digits in the star number). The content of each record is as follows:

    RA, western limitStored as Single
    RA, eastern limit stored as Single
    Dec, Northern limit stored as Single
    Dec, Southern limitstored as Single
The units for all co-ordinates are degrees.

To locate a specific star in a Gaia catalogue from its Tycho-2 number:

  • use the Region number (the first element in the Tycho2 number) to extract from this file the limits in RA and Dec for that Tycho2 region.
  • for the Gaia file being accessed, use its corresponding .inx to obtain the range of records to be scanned in the 0.5° bands of that Gaia catalogue file.
  • Scan the Gaia file to find an entry with the required Tycho2 number

Not all stars with a Tycho2 number will be found. There are two reason:

  1. Each Gaia entry contains on a single star identifier. A Hipparcos number has precedence over a Tycho2 number, and as a result any Tycho2 stars which are also in Hipparcos cannot be located using the Tycho2 number.
  2. The Tycho2 numbers in the Gaia catalogues have been obtained via the TychoGaia file of Gaia DR1. Beyond that, no effort has been made to match Gaia stars to the original Tycho2 catalog.

NOTE: The Tycho2 star numbers within a Region are not ordered by Right Ascension or Declination.
