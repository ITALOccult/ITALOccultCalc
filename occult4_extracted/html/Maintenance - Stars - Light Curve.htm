AAVSO Light Curve

If a star is a variable star, Occult provides the ability to retrieve a light curve using the AAVSO Light Curve Generator.

When Occult accesses the light curve generator, it does not use all the parameters that are available. Anyone wanting to vary the full range of parameters should directly access the AAVSO light Curve Generator . When retrieving a light curve, the parameters that can be set are:

  • The variable identifier. This will usually be pre-set when making the query. However the user can specify any variable star identifier. The format required is not case sensitive. However it does require the presence of a space in the identifier. Examples of allowable identifiers are:
    • R Sco
    • V1190 Sco
    • NSV 7477
    • alf Sco [= alpha Sco. Care is required for variables identified by a Bayer letter]
    • ASAS J160849-8207.8

  • The start period for the data to be plotted. The default start date is January of 5 years before today's date. The plot includes all available observations since that date.

  • The data to be included in the plot. Five categories are available:
    • U : Observations made in the photometric U band
    • B : Observations made in the photometric B band
    • V : Observations made in the photometric V band
    • R : Observations made in the photometric R band
    • Visual : Observations made visually
    The default selection is to include V, R and Visual observations.
To retrieve the light curve, click Get light curve. The form will indicate that it is retrieving data from the AAVSO web site.

If a light curve is available, the form will display the light curve, which can then be Copied, or Saved, from the with Light Curve menu item.

If a light curve is not available, the form will display the explanation provided by the AAVSO Light Curve Generator.

NOTE.The majority of variable stars will not have a light curve available from the AAVSO Light Curve Generator.

If several light curves are retrieved, buttons are provided to step back and forward between the light curves.
