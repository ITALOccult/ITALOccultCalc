Double star report

A lunar occultation of a double star, when observed from significantly different Position Angles against the moon, can allow for an accurate measurement of the Position Angle and separation of the double star. Occult supports the easy reporting of observations for this purpose.

The functionality is contained in the menu of the Lunar Observations editor - as Double star report... Create report for current star. The functionality is only enabled when the Events page of the editor is displayed. The functionality can be used by selecting an event already entered into the editor. It can also be used for an event partly entered - assuming the date, star number, Station and Observer have been entered/selected.

On clicking Double star report... Create report for current star (brighter component), the program will open the Double Star reporting form. It will populate the form with a range of data relevant to the selected event. Some of this is shown on the form. Not shown are the station and observer details, and the catalogue details of the double star.
IMPORTANT. When creating a double star report, always base it on the brighter component of the pair. If you can't decide which is brighter, use the first event.

Data entry

Event times

Most of the data requred for the report is populated when the report is created. This includes the hour and minute of the occultation, as well as the seconds of the Bright star that is used to create the report. The one data item requiring manual entry is the time difference in seconds for the fainter (second) star.

The occultation of the fainter star can be either before or after the brighter star. If it occurs earlier, the time differenbce will be negative. When you enter the time difference, the computed seconds of the event are shown below the box - as a verification that you have entered the time difference correctly. When the event spans a minute, the value displayed under the box is just of the final seconds. For example, let us assume events occurring at 23m 59.3s, and 24m 01.2sec. If the brighter star was the first star, the time difference would be entered as +1.9, and the value displayed under the box will be 1.2 (not 61.2). Conversely if the brighter star was the second star, the time difference to be entered would be -1.9, and the time displayed below the box will be 59.3 (not -0.7).

LiMovie Light Curve

If you have recorded the occultation by video, you should generate a light curve using the software LiMovie. When you have measured the video, you can plot the lightcurve within LiMovie. At the bottom of the form displaying the light curve, there is a button Copy to clipboard. Click that button.

Now on the Double Star Report form, click Retrieve LiMovie plot from the Clipboard. This will place the LiMovie graphic into the space on the report form.

You can use this functionality to copy different images into the form. For example, LiMovie can plot a 3-D image of the region surrounding the star, which can be copied to the clipboard, then pasted into the double star report form. However only one graphic can be included.

Additionally, having brought the LiMovie light curve into the report form, you can save it using any of the 5 image formats supported by Occult.

Message

A large box is provided at bottom left, where you can enter any message you want to accompany the report.

Email addresses

The Email address for sending the report is downloaded from the IOTA web site. This will be automatically updated if you are connected to the internet when you start Occult. Otherwise you may need to update the file of Addresses from teh General Downloads page of Occult, under item #10. The address is shown at bottom right - immediately after the area for the light curve graphic. By default that address is not editable, and the box is coloured red. Clicjk the 'Edit Address' next to the box,and the color will change to Green, and you can edit the email address. However the edited email address will not be retained. You can also copy the email to multiple email addresses - which you specify in the box at bottom right. Rules for formatting the addresses are:
  • The actual address is held within angled brackets. Example... <Fred@hisDomain.com>
  • As usual, upper and lower case does not affect the email address
  • You can precede the email address with an identifier - held between double-inverted-commas. Example... "Fred is fantastic"<Fred@hisDomain.com>
  • Multiple email addresses must be separated with a semi-colon ( ; ). Example... <Bill@iinet.com.au>;"Fred is fantastic"<Fred@hisDomain.com>;"<Joe@Bigpond.com>
  • Commas, colons, and carriage returns are not allowed (and cannot be entered).

Recognising that reports will typically be sent to the same group of people, the CC email list can be saved as a default list - using the button to the right of the email address block. If you edit the displayed list of email addresses, you can re-insert the default addresses - again using the button to the right of the email address block.

Viewing, Sending and Saving the report

You can view the text content of the report and any message by clicking Preview report.

The report (with comments) is Emailed to the email addressees by clicking Save and Email double star report. Before attempting to email the report, and any light curve, will be saved. The default directory is \Occult 4\Observations\Doubles, and the default file name is made up of the date and star number. Then the report together with any email message will be emailed.

If you only want to save the report, click Save double star report. This will save the report and any light curve. This may be useful if you need to email the report using your usual email software.
