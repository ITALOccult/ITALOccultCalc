AsteroidElements.csv contains the orbital elements of asteroids. It is used for computing predictions of occultations by asteroids.

UserMinorPlanetElements.csv contains the orbital elements of a select number of asteroids or comets for use in asteroid occultation predictions. The elements can be imported from AsteroidElements.dat, or manually entered.

UserMinorPlanetElements_Lunar.csv contains the orbital elements of a select number of asteroids or comets for use in Lunar occultation predictions. The elements can be imported from AsteroidElements.dat, or manually entered.

The format of all three files is the same. The elements are referred to the J2000 equinox.:

These files, introduced in 2018, contain the elements in a csv format - thereby allowing variable precision in the elements held in the file.

The first line of the file is a header identifying the fields. The header line is:

Num,Name,year,month,day,MA,perih,node,i,e,a,H0,G,Dia,DiaSrc,PEU,#Rings,#Moons,OrbSrc,OrbDate,Class

AsteroidElements.dat contains the orbital elements of asteroids. It is used for computing predictions of occultations by asteroids.

UserMinorPlanets.dat contains the orbital elements of a select number of asteroids or comets for use in asteroid occultation predictions. The elements can be imported from AsteroidElements.dat, or manually entered.

UserMinorPlanets_Lunar.dat contains the orbital elements of a select number of asteroids or comets for use in Lunar occultation predictions. The elements can be imported from AsteroidElements.dat, or manually entered.

The format of all three files is the same, and have a record length of 145 characters. The elements are referred to the J2000 equinox. The format is:

Columns Meaning
0 - 6 Asteroid number
8- 23 Asteroid name
24-38 The osculating date for the elements or the date of perihelion. Format is 24-28 year, 29-30 month, 31-38 day. Day is an integer if the elements specify the mean anomaly, and has up to 5 decimal places if the elements use the date of perihelion.
39-49 MA - Mean anomaly at the date of the elements. Set to '0' if the elements specify the date of perihelion with no mean anomaly.
50-60 ϖ - Argument of perihelion (deg)
61-71 Ω - Longitude of ascending node (deg)
72-82 i - Inclination (deg)
83-93 e - eccentricity of orbit
94-106 a - semi-major axis; if e>0.97, q - perihelion distance
107-112 H0 - magnitude constant
113-118 G - Slope parameter in the magnitude equation for asteroids. Commonly 0.15 For comets, set to '0'
119-125 Max diameter, in km
126-132 Min diameter, in km. Left blank if there is no determination.
133-134 Diameter source [1=IRAS mimpsalb, 2=IRAS diamalb V6, 3=IRAS single V6, A = AKARI AcuA, B= WISE, C = TNO & Centaurs (from PDS)]
[The following are not used as of 2016: 0=Unknown, 4=Richmond, 5=Photometry, 6=from H0, 7=Occultation, 8=Direct imaging, 9=Miscellaneous]
