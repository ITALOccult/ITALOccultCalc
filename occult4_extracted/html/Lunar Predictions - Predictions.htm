Lunar Occultation Predictions

To generate occultation predictions, you need to:
  • Select the site for the predictions details
  • Select the star catalogue to use details
  • Select the objects to include in the predictions details
  • Set the date details; and
  • Generate the predictions details.

The details given in ORDINARY occultation predictions are explained here.

The details given in GRAZING occultation predictions are explained here

When predictions have been generated, additional functions are available by right-clicking on a prediction line. The following sets out the right-click functions that are available for ordinary occultations, and grazing occultations

Multi-Site prediction

Generates predictions of the selected star for all sites in the current site file. Multisite predictions are usually generated for publishing predictions of special events in the media.

Predictions are only given if the occultation meets the visibility requirements for the particular sites. The Multi-Site prediction has either two or three blocks:

  • The first block [optional] provides predictions which include PA, WA and the differential corrections a and b.
  • The second block [optional] has merely the location, UT, altitude of sun and Moon, and Cusp angle.
  • The third block gives the coordinates of each site that appears in the predictions.

You can set the output to include either, or both, the first and second blocks, using the output button below the Multi-site button. The settings are retained between sessions.

Disappearance events are listed separately from Reappearance events. If the D or R phase does not meet the visibility requirements for any site, the prediction will make no reference to that phase.

The Multi-Site listing will include events involving bright stars that occur in daytime. Such events can be excluded from the listing by a User Setting (under the Maintenace tab). In the box for 'Lunar Occultation Predictions - Grazes, Multisite predictions', check the box at the bottom - 'Exclude ALL events with a sun altitude <-6 deg.'

When a Multi-Site prediction has been computed, the right-click menu provides an option to plot a moon map showing the star paths for all sites, and to re-list the ordinary occultations for that day.

World mapDraws the occultation region on a Mercator projection of the world. The plot indicates various regions in the following way:
  • Night-time - white continuous line;
  • Twilight - blue continuous line (not distinguished from night-time in B&W plots)
  • Daylight - red dotted line
  • The region where only one of D or R is visible because of an intervening moonrise or moonset are indicated by ellipses at the end of the occultation path.
If the path is near the poles the occultation path can appear quite distorted due to the Mercator projection.
View in GoogleEarth If GoogleEarth has been installed on the computer, clicking this will display the occultation path on GoogleEarth. This means you can use all the facilities of GoogleEarth to investigate possible observing locations.
Save GoogleEarth kmz fileClicking this will create a '.kmz' file that can be used to display the occultation path in GoogleEarth. [e.g. you could email the path to a friend for them to view on their computer.]
Save GoogleMaps HTM fileClicking this will create an '.htm' file for use with GoogleMaps to display the occultation path. This option is for people who run a web page to publicise events, enabling them to easily create a web page for the event.

Weather predictions

Occult can access the APanel component of the 7-Timer numeric weather prediction system - by clicking on the menu item 3-day weather forecast..

If ordinary occultations are displayed (or no other predictions have been generated), cloud coverage and temperature (Centigrade) are displayed at 3-hour intervals. The prediction is for the next 3 days at the location of the currently selected site.

If a graze prediction has been generated, and the prediction is less than 3 days in the future, cloud coverage and temperature (Centigrade) are displayed for locations along the path. If there are more than 20 prediction lines, the weather details are not obtained for each point - as it takes a few seconds to retrieve the data for each point from 7-Timer.
