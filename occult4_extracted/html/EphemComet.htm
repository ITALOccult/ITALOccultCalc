Comet ephemeris

This routine generates ephemerides of all currently-observable comets using the file of comet orbital elements [comet.dat] maintained by the IAU Minor Planet Center.

In order to generate an ephemeris, you must have a copy of comet.dat. To do this, click the 'Download Comet.dat' button. After a few seconds the file should download. Download this file regularly to make sure that all the latest comets are included.

NOTE: If you have trouble getting a connection to download MPCOrb, read MPCOrb database, conduct the test of whether your ISP has properly registered their DNS address - and if not, complain to your ISP. [This is a security issue implemented by Harvard University; ISP's that don't register their DNS are typically used by spammers. If your ISP does not Register its DNS, you might consider changing your ISP.]

To generate predictions:

  • select the comet using the drop-down list;
  • indicate whether you want the orbital elements displayed with the prediction
  • set the start day and hour for the prediction. Note: set the hour (UT) so that the ephemeris is computed for the time of day that you will be viewing the comet.
  • set the duration of the ephemeris, in days
  • set the ephemeris interval in days.

    Note that the ephemeris is computed on the basis of the osculating elements without any numerical integration of the orbit. You should limit predictions to a year to avoid significant 'errors' in the computed position.
